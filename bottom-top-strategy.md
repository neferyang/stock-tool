# 底部 / 頭部確認與操作策略參考手冊

## 底部確認完整清單

### 技術訊號（至少出現3個才算確認）

#### 必要條件（最重要）
- [ ] 週K線出現止跌，收盤高於前週最高價
- [ ] 成交量型態：連續萎縮後出現放量長紅K

#### 輔助條件
- [ ] KD值在20以下黃金交叉向上
- [ ] MACD柱狀體由負轉正（死叉轉金叉）
- [ ] RSI從30以下回升突破40
- [ ] 布林通道下軌出現支撐，股價回彈
- [ ] 出現錘子線或晨星K線組合
- [ ] 連續下跌後出現「價跌量縮→價漲量增」的轉換
- [ ] 費波南希61.8%回撤位附近獲得支撐

#### 籌碼面輔助
- [ ] 外資由連續賣超轉為買超
- [ ] 投信開始買進（投信通常是底部先行指標）
- [ ] 融資餘額大幅下降（散戶被洗出，籌碼乾淨）

---

## 頭部確認完整清單

### 技術訊號（至少出現3個才算確認）

#### 必要條件（最重要）
- [ ] 週K線出現高點後連續兩週收黑
- [ ] 量價背離：股價創新高但成交量萎縮

#### 輔助條件
- [ ] KD值在80以上死亡交叉向下
- [ ] MACD出現高檔背離（股價新高，MACD未新高）
- [ ] RSI從70以上下滑跌破60
- [ ] 布林通道上軌無法突破，股價壓回
- [ ] 出現墓碑線或暮星K線組合
- [ ] 竭盡缺口出現後短期回補
- [ ] 長上影線頻繁出現（上方賣壓沉重）

#### 籌碼面輔助
- [ ] 外資由買超轉為賣超
- [ ] 董監事申報轉讓（內部人出脫持股）
- [ ] 融資餘額大幅增加（散戶追高，過熱警訊）

## 進出場策略框架

### 積極型策略（適合短線操作）

**進場條件：**
- 底部訊號出現2個以上
- 進場價：突破前一日最高價
- 停損：設在進場價-3%至-5%

**出場條件：**
- 達到目標價（費波南希延伸位）
- 或頭部訊號出現2個以上

---

### 保守型策略（適合穩健投資人）

**進場條件：**
- 底部訊號出現4個以上確認
- 週K線確認反轉（連續兩週收紅）
- 進場價略高但安全性高

**出場條件：**
- 週K線連續兩週收黑
- 或跌破重要支撐（季線/年線）

---

### 分批布局策略（適合存股/中長線）

**適用情境：**
- 基本面良好但股價仍在下跌
- 不確定底部確切位置時

**執行方式：**
```
第一批（30%倉位）：在第一支撐位買進
第二批（30%倉位）：在第二支撐位買進
第三批（40%倉位）：底部確認訊號出現後買進

停損：整體持倉虧損超過10%需重新評估
```

---

### 存股策略（適合長期持有）

**進場條件：**
- 殖利率達到個人要求（通常4%以上）
- 本益比低於歷史平均（本益比河流圖藍色區間）
- 公司基本面穩健（EPS連續成長/穩定）

**持有邏輯：**
- 不看短期股價波動
- 關注每年配息
- 逢低加碼（攤平成本）

## 常見錯誤與避免方法

| 常見錯誤 | 正確做法 |
|---------|---------|
| 摸底太早（底部未確認就進場） | 等待至少3個底部確認訊號 |
| 追高（看到上漲才進場） | 在回撤至支撐位才進場 |
| 沒有停損計畫 | 進場前先設好停損點 |
| 只看一個指標 | 多指標共振確認才操作 |
| 過度自信預測底部 | 承認無法精準抓底，採分批策略 |
| 忽視大盤趨勢 | 個股逆大盤成功率低，先看大盤 |
