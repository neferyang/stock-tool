## 🔧 手動更新流程

### 步驟 1：查詢單支股票的財務數據

**訪問網址：** https://mops.twse.com.tw/mops/web/t05_query

1. 搜尋股票代號（例如 `2330`）
2. 頁面會顯示該公司的財務報表
3. 找到以下欄位，記錄 **最近 5 年** 的數據：
   - **EPS**（每股盈餘）
   - **ROE**（股東權益報酬率）
   - **淨利率**
   - **營業利率**
   - 其他需要的財務指標

### 步驟 2：編輯 eps-correction-template.json

**位置：** `eps-correction-template.json`

**範例格式：**
```json
{
  "corrections": {
    "2330": {
      "name": "台積電",
      "issue": "補充完整的 5 年財務數據",
      "historicalEPS": [
        {
          "year": "2021",
          "eps": 13.05,
          "source": "TWSE MOPS"
        },
        {
          "year": "2022",
          "eps": 15.52,
          "source": "TWSE MOPS"
        },
        // ... 繼續添加 2023, 2024, 2025
      ]
    }
  }
}
```

### 步驟 3：執行數據合併

```bash
python3 apply-eps-correction.py
```

### 步驟 4：推送更新

```bash
git add eps-correction-template.json index.html
git commit -m "update: 更新 TWSE MOPS 財務數據

- 股票代號: 補充 5 年 EPS 和財務指標
- 數據來源: TWSE MOPS 公開資訊觀測站
- 驗證日期: YYYY-MM-DD"

git push origin main
```

## 📊 TWSE MOPS 查詢指南

### 如何找到 EPS 和其他指標

在 MOPS 的查詢結果中：

| 欄位 | 在哪找 | 單位 |
|------|--------|------|
| **EPS** | 個股月營收/年度財報 | 元 |
| **ROE** | 財務比率分析 | % |
| **淨利率** | 財務比率分析 | % |
| **營業利率** | 財務報表 | % |
| **負債比** | 財務比率分析 | % |

### 查詢技巧

- **快速查詢**：輸入股票代號後按 Enter，不要按「查詢」按鈕
- **年度數據**：優先選擇「年度」而非「季度」
- **完整年份**：記錄完整 5 年（2021-2025）
- **四捨五入**：EPS 通常保留 2 位小數，百分比保留 1 位

## 💡 自動化備選方案

1. **定時爬蟲**（需要代理或延遲控制）

3. **用戶上傳**
   - 提供 Web 表單，用戶上傳 CSV 或 JSON
   - 結合手動查詢和自動導入

## 📞 常見問題

**Q: 為什麼不能自動爬蟲？**
A: MOPS 有反爬蟲防護，自動請求會被阻止。手動查詢速度快、可靠。

**Q: 更新頻率如何？**
A: 建議每季度（3 個月）更新一次，或在重要股票發佈季報時立即更新。

**Q: 可以只更新 EPS 嗎？**
A: 可以。但完整的財務指標（ROE、淨利率等）對診斷卡更有用。

**Q: 如何驗證數據正確性？**
A: 在 MOPS 上查詢的同時，也可在公司官網或證交所網站驗證數據。
