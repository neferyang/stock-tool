### **驗證階段（10 分鐘）**

#### **步驟 1：清除快取**
```
Ctrl + Shift + R  (Windows/Linux)
Cmd + Shift + R   (Mac)
```

#### **步驟 2：測試搜尋**
- [ ] 搜尋 **2330 台積電** → 檢查診斷卡 ROE、淨利率等是否正確
- [ ] 搜尋 **2207 和泰車** → 檢查數據
- [ ] 搜尋 **2454 聯發科** → 檢查趨勢圖年份順序

#### **步驟 3：驗證年份順序**
- [ ] 5 年趨勢圖應該是 **2021 → 2022 → 2023 → 2024 → 2025**（從左到右）
- [ ] 不是 2025 → 2024 → ... → 2021 的反向
