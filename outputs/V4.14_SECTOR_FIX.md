# v4.14 Sector 分類修復

**修復日期**：2026-06-04  
**問題**：AAPL 等美股仍顯示「未分類」而非「電腦及週邊設備業」  
**根本原因**：Yahoo Finance 可能不在 price 對象中返回 sector 信息

---

## 🔧 修復方案

### 1. 改進 Sector 數據提取

**位置**：summaryResult 數據整合部分

```javascript
// 修改前
if (!m.sector) m.sector = raw2(pr.sector)||'';

// 修改後：從多個位置查找 sector
if (!m.sector) m.sector = raw2(pr.sector) || raw2(pr.quoteType) || raw2(sd.sector) || '';
```

**目的**：增加 sector 數據查找範圍

---

### 2. 新增英文公司名稱匹配（L1.5 級）

**位置**：classifyIndustry() 函數中，在 L2 之前

```javascript
const enCompanyMatches = [
  // 科技
  ['apple|microsoft|nvidia|adobe|intel|amd|meta|google|amazon', '電腦及週邊設備業'],
  
  // 金融
  ['jpmorgan|goldman|citigroup|bofa', '金融保險業'],
  
  // 醫療
  ['pfizer|johnson|jnj|eli lilly|merck|moderna', '生技醫療業'],
  
  // 能源
  ['exxon|chevron|shell|bp', '油電燃氣業'],
  
  // 零售
  ['walmart|costco|target|home depot', '食品工業'],
  
  // 汽車
  ['tesla|ford|gm|general motors', '汽車工業'],
  
  // 其他
  ['coca|pepsi|nestl', '食品工業']
];

for (const [pattern, officialName] of enCompanyMatches) {
  if (new RegExp(pattern).test(nm)) {
    return {industry: officialName, confidence: 'high', details: {method: '英文公司名稱', name}};
  }
}
```

**效果**：
- ✅ AAPL（Apple）→ 電腦及週邊設備業
- ✅ MSFT（Microsoft）→ 電腦及週邊設備業
- ✅ JNJ（Johnson & Johnson）→ 生技醫療業
- ✅ INTC（Intel）→ 電腦及週邊設備業
