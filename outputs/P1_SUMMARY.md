## 🔧 技術實現

### newsAnnouncementsFetch() 函數

**位置**：index.html，第 192-300 行

**邏輯流程**：
```
1. 方案 A：TWSE API
   ├─ fetch(mops_t187ap05_${code})
   ├─ 8 秒超時
   └─ 成功 ✓ 返回，失敗 ✗ 進入方案 B

2. 方案 B：MOPS 爬蟲
   ├─ fetch(mops.twse.com.tw/mops/web/...)
   ├─ HTML 正則解析
   ├─ 提取表格數據
   └─ 成功 ✓ 返回，失敗 ✗ 進入方案 C

3. 方案 C：回退連結
   ├─ 返回提示項目
   ├─ 提供 MOPS 直接連結
   └─ 用戶可手動查詢
```

### 公告分類邏輯

**主要變數**：
```javascript
m.news = [{date, title, type, url}, ...]
m.majorNewsCount    // 重大訊息數
m.minorNewsCount    // 常規公告數
m.newsCount         // 總公告數
m.newsSource        // 數據來源
```

**重大訊息判定**：
```javascript
type === 'M' 
  || title.includes('重大')
  || title.includes('公開發行')
```

### 決策信號融合

**邏輯**：

### 核心原則

1. **優先級明確**

   - 常規公告次要（灰色）

   - 三層備援確保可用
