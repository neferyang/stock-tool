## 🔍 詳細測試步驟

### 測試 1：AAPL 美股警告

```
1. 輸入「AAPL」→ 點擊「分析」
2. 檢查「基本面數據」上方
3. 確認顯示黃色警告框
4. 警告框包含：
   - ⚠️ 符號
   - 「美股數據限制」標題
   - 說明文字
5. 基本面數據區域顯示 N/A
```

✓ **通過** / ❌ **失敗**

### 測試 2：MSFT 產業分類

```
1. 輸入「MSFT」→ 點擊「分析」
2. 查看「基本面數據」中的「產業分類」
3. 應顯示「數位雲端 (high)」
4. 不應顯示「未分類」
```

✓ **通過** / ❌ **失敗**

### 測試 3：2330 台股正常

```
1. 輸入「2330」→ 點擊「分析」
2. 確認：
   - 不顯示美股警告框
   - 基本面數據有值（P/E、P/B 等）
   - 產業分類為「半導體業」
```

✓ **通過** / ❌ **失敗**

### 測試 4：SPY ETF 正常

```
1. 輸入「SPY」→ 點擊「分析」
2. 檢查產業分類
3. 應顯示「ETF」
4. 不應顯示警告框（ETF 無基本面數據正常）
```

✓ **通過** / ❌ **失敗**
