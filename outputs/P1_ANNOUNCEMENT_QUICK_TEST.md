### 步驟 2：查看公告卡片

查詢後向下滾動，找到 **「📰 公司公告」** 卡片

**應該看到**：
```
📰 公司公告

🔴 重大訊息    📌 一般公告    ⏰ 最新公告    🔗 詳細查詢
  N 則          N 則          2026-06-XX      MOPS →
需特別關注

最新：公告標題摘要...
```

**若看不到公告卡片**：
- ❌ 可能 API 請求失敗
- ✅ 檢查瀏覽器 Console（F12）的日誌
- ✅ 查看是否有 [公告] 開頭的日誌信息

---

### 步驟 3：檢查 Console 日誌

**打開瀏覽器 Developer Tools**（按 F12）

**查看 Console 標籤**，應該看到類似日誌：

#### 成功案例
```
[公告] 正在查詢 2330 的最新公告...
[公告] ✅ TWSE API 成功取得 10 則公告
或
[公告] ✅ MOPS 爬蟲成功取得 10 則公告
```

#### 失敗與回退案例
```
[公告] 正在查詢 2330 的最新公告...
[公告] TWSE API 失敗: timeout
[公告] 使用備援方案，爬取 MOPS 頁面...
[公告] MOPS 爬蟲失敗: Network error
[公告] ⚠️ 使用備援方案
```

**預期結果**：
- ✅ 至少有一條成功日誌
- ✅ 或成功回退至備援方案
- ✅ 應用不會卡住或拋錯

---

### 步驟 4：驗證公告分類

**檢查卡片中的計數**：

```
若顯示：
  🔴 重大訊息 3 則
  📌 一般公告 10 則
  
則說明：
  ✅ 公告分類正常工作
  ✅ 重大訊息被正確識別
```

**檢查公告來源**（進階）：

在 Console 執行以下命令查看數據：
```javascript
// 查看最新查詢結果中的公告數據
console.log('公告數據:', {
  news: m.news,
  majorCount: m.majorNewsCount,
  minorCount: m.minorNewsCount,
  source: m.newsSource
});
```

**預期輸出**：
```javascript
{
  news: [
    {date: "2026-06-04", title: "...", type: "M"},
    {date: "2026-06-03", title: "...", type: "C"},
    ...
  ],
  majorCount: 3,
  minorCount: 10,
  source: "TWSE"  // 或 "MOPS" 或 "FALLBACK"
}
```

### 步驟 5：驗證決策信號中的公告警示

**查看買/賣/持建議**（綠色卡片區域）

**若公司有重大訊息，應該看到**：

```
🔴 估值偏低，可考慮買進 (🔴有重大訊息，建議謹慎)

多個估值模型顯示股價低於合理價值...
但請先確認重大訊息內容。
52週低點 XX 為重要支撐...

📰 最新公告 (2026-06-04)：公告標題...
⚠️ 注意：本公司有 N 則重大訊息，建議查詢公開資訊觀測站
瞭解詳細內容，再做決策。
```

**驗證項目**：
- ✅ 標題包含 (🔴有重大訊息) 標籤
- ✅ 正文包含警告文字
- ✅ 公告摘要和日期顯示
- ✅ 警告措辭清楚

## ⚠️ 常見問題

### Q1: 為什麼看不到公告卡片？

**可能原因**：
1. API/爬蟲都失敗，使用回退連結
2. 股票代碼不存在或無公告
3. 網路連接問題

**解決方案**：
- 檢查 Console 日誌（F12）
- 查看是否有 [公告] 開頭的日誌
- 嘗試更換股票重新查詢
- 檢查網路連接

### Q2: 為什麼公告數量為 0？

**可能原因**：
- 該股票確實無公告
- 新上市企業尚未有公開訊息
- API 返回空結果

**解決方案**：
- 這是正常情況，應用會提供 MOPS 連結
- 用戶可手動在 MOPS 查詢
- 切換其他股票測試

### Q3: Console 出現錯誤但應用仍正常？

**這是正常的!** 公告功能已實現：
- ✅ 錯誤被正確捕捉
- ✅ 自動降級到備援方案
- ✅ 應用主功能不受影響
- ✅ 用戶無感知故障

### Q4: 重大訊息標籤怎樣才能出現？

**需滿足以下條件之一**：
1. API 返回 `type: 'M'` 的記錄
2. 公告標題包含「重大」關鍵字
3. 公告標題包含「公開發行」關鍵字

**測試**：
- 選擇公告較多的公司（如台積電）
- 查看是否有紅色 🔴 標籤
- 檢查公告摘要是否包含「重大」
