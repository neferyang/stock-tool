# Toyota 經銷商教育訓練 Plugin

完整的教育訓練資料庫與講義生成工具，集成了 300+ 個教育訓練教材。

## 功能概覽

### 📚 教訓練講義生成
快速生成教案大綱、完整講義或 PPT 簡報框架
- 指定課程主題、目標學員、訓練時長
- 自動產出結構化教案
- 包含教學活動、補充資料、講師提示

### 🔍 教育資料查詢
智慧檢索 300+ 個教育訓練教材
- 按課程類別、等級、關鍵字搜尋
- 提供教材清單、版本標示、搭配建議
- 推薦最新版本和學習順序

### 📋 訓練計畫設計
規劃跨據點的訓練進度
- 內建課程組合建議（3 個月計畫、5 天計畫等）
- 進度安排和評估方法
- 確保訓練內容一致性

## 涵蓋課程

**業代教育**
- 新進業代基礎訓練（TWSM、商品知識、客戶關係等）
- 中級業代進階課程
- 嘉信新進業代專門教材（分期租賃、保險等）
- 各車款商品知識（ALTIS、CAMRY、YARIS、Prius、Previa、Innova 等）

**主管教育**
- 新任課長管理基礎課程
- 營業所管理教育（時間管理、PDCA 等）
- 儲備課長晉升訓練
- 豐田管理工具與理念

**初級 OJT**
- 新進人員現場訓練指引
- 展示室應對、交車說明、中古車查估

**補充資料**
- 管理故事與案例教材
- 激勵與態度教育
- Toyota 企業文化與哲學

## 快速開始

告訴 Claude 你的訓練需求，Skill 會自動觸發：

```
「我需要新進業代的 TWSM 豐田銷售方式 2 小時教案大綱」
「列出所有主管管理課程」
「產出客戶關係管理的 30 分鐘 PPT 簡報版本」
```

## 使用範例

### 場景 1：快速備課
```
「新進業代、2 小時、TWSM 豐田銷售方式、教案大綱格式」
↓
輸出：課程目標、3 個主要單元、時間配置、教學活動、補充資料
```

### 場景 2：查詢教材
```
「列出所有 CR（客戶關係管理）相關教材」
↓
輸出：新進 + 進階課程清單、版本標示、訓練路徑、推薦順序
```

### 場景 3：規劃訓練計畫
```
「設計新進業代的 3 個月訓練計畫」
↓
輸出：課程組合、進度表、評估方法、KPI 指標
```

## 輸出格式

- **教案大綱** — 課程目標、主要單元、時間配置、教學要點
- **完整講義** — 詳細的講授內容、案例、練習、補充資料
- **PPT 簡報框架** — 視覺化指導、講師提示、互動設計
- **課程清單** — 教材清單、分類、版本狀態、搭配建議

## 常見問題

**Q: 能否調整生成的教案內容？**
A: 完全可以。直接告訴 Claude 想修改的部分，會自動調整。

**Q: 資料庫的教材可以直接教學嗎？**
A: 可以。Skill 提取的是現有教材內容，生成的教案適合直接在課堂使用。

**Q: 找不到需要的課程怎麼辦？**
A: 告訴 Claude 你的需求，會根據現有教材組合新課程或推薦替代方案。

**Q: 輸出格式可以轉換嗎？**
A: 預設是 Markdown 或 PPT 框架，可要求轉換為 PDF、Word 或其他格式。

**Q: 能否客製化課程內容？**
A: 可以。提供具體需求（人員等級、訓練目標、時間限制），會客製化課程。
