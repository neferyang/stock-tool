## 📌 使用指南

本指南整合了台灣證券交易所各產業的法人評價重點，幫助投資者快速判斷股票是否被低估或高估。

### 核心概念

| 估值狀態 | P/E 範圍 | 買賣信號 | 說明 |
|---------|---------|---------|------|
| **低估** | < fairPE × 0.8 | 🟢 買進 | 具吸引力，可分批建倉 |
| **合理偏低** | fairPE × 0.8 ~ 1.0 | 🟡 觀察 | 值得長期持有 |
| **合理** | fairPE × 0.8 ~ 1.3 | 🟡 持有 | 合理估值區間 |
| **合理偏高** | fairPE × 1.0 ~ 1.3 | 🟡 觀望 | 可逐步減碼 |
| **高估** | > fairPE × 1.3 | 🔴 賣出 | 風險較高，謹慎操作 |

### 景氣循環股特殊邏輯

**航運、鋼鐵、塑膠、化學等原物料相關**：
- 景氣大好時：獲利爆發但 P/E 反而極低 → 反而應該謹慎
- 景氣谷底時：獲利低迷但 P/E 高 → 可考慮布局
- 法人多以股價淨值比（P/B）作為底部支撐指標

## 🎯 實際應用案例

### 案例 1：台積電 (2330) - 半導體業

```
查詢結果：
- 目前股價：630 元
- P/E 比：28x
- 產業 fairPE：25x
- 合理本益比區間：25x × 0.7 ~ 25x × 1.3 = 17.5x ~ 32.5x

評估：
✓ P/E 28x 落在 17.5x-32.5x 合理區間內
✓ 相對 fairPE 略高（28/25 = 1.12x）→ 合理偏高
✓ 建議：持有或逢低加碼

合理股價估計：630 ÷ 28 × 25 = 562.5 元（合理價值）
```

### 案例 2：中鋼 (2002) - 鋼鐵工業

```
查詢結果：
- 目前股價：32 元
- P/E 比：8x
- 產業 fairPE：10x
- 合理本益比區間：10x × 0.7 ~ 10x × 1.3 = 7x ~ 13x

評估：
✓ P/E 8x 低於 fairPE 10x
✓ 落在低估偏低區間（8/10 = 0.8x）
✓ 但考慮景氣循環，可能處於獲利預期高點
✓ 建議：分批布局，參考股價淨值比（P/B）

合理股價估計：32 ÷ 8 × 10 = 40 元（合理價值）
底部支撐：若 P/B = 1.0x，根據淨值評估
```

## 💡 進階應用技巧

### 1. 四象限分析法

```
       P/E 相對合理性
        
        高估象限    正常象限
        (減碼區)    (持有區)
            |
    ────────┼──────→ fairPE
            |
    低估象限    谷底象限
    (買進區)    (逆向區)
```

### 2. 景氣循環股操作

**逆向邏輯**（航運、鋼鐵、原物料）：
- 高 P/E（相對歷史）→ 景氣末期 → **減碼賣出**
- 低 P/E + 高 P/B → 景氣谷底 → **布局買進**

### 3. 產業對比法

同族群內產業比較：
```
高成長族群內比較：
- 如果半導體 P/E 25x，數位雲端 P/E 28x
- 數位雲端 fairPE 30x（高於半導體 25x）
- 則數位雲端相對偏低估，值得看好
```

## ⚠️ 使用注意事項

1. **fairPE 只是參考**
   - 不同公司在同產業內估值差異大
   - 需結合公司基本面、成長性、市場地位

2. **景氣循環股需特殊處理**
   - 不能簡單看 P/E
   - 需要看景氣週期位置和 P/B

3. **新興產業評價波動大**
   - 如綠能、生技、數位雲端
   - fairPE 可能需要定期調整

4. **金融股看 P/B 優於 P/E**
   - 銀行、保險應以 P/B 1.0-1.5x 為基準
   - P/E 只作為輔助參考

5. **還需考慮其他因素**
   - 股利殖利率（防禦型）
   - 股價淨值比（景氣循環）
   - 成長率（高成長）
   - 技術面訊號（買賣時機）
