# 業務全流程養成計畫 — 客源開發·客戶維護·社群媒體·來電客經營

## 課程基本資訊

- **課程名稱**：汽車業務代表全流程業務養成計畫
- **訓練對象**：Toyota 經銷商新進業代（入職後 3 週以上）
- **訓練時數**：12 週（3 個月），每週 4-6 小時
- **課程等級**：初級～進階
- **課程類別**：業務技能核心課程

---

## 課程概述

### 為什麼需要業務全流程養成？

在傳統的業務訓練中，我們通常只教業代「如何完成一筆銷售」。但實際上，一名優秀的業代需要做的遠遠不止這些：

- **主動開發客源**，而不只是等客上門
- **主動維護客戶**，確保客戶的續保和轉介
- **會運營社群媒體**，建立個人品牌和客戶粘性
- **會經營來電客**，把打進來的電話轉化為客戶

這四個環節形成了一個**完整的業務生態循環**：

```
客源開發 → 客戶開拓 → 成交 → 客戶維護 → 社群經營 → 轉介和續保
  ↑                                        ↓
  └────────────────────────────────────────┘
```

**核心問題**：許多業代停留在「被動接待客戶、完成成交」的階段，而優秀業代則主動控制客戶生命週期的每一個環節。

### 課程的三大目標

1. **掌握四大業務環節** — 從客源開發、到銷售成交、再到客戶維護和社群運營，形成閉環
2. **建立個人品牌和客戶資產** — 不是「為公司賣車」，而是「為自己積累客戶」
3. **達成業績和滿意度的雙贏** — 3 個月內月均業績達 60+ 萬，客戶滿意度 85+ 分

## 核心模組

### 【第一模組】客源開發策略與執行（第 1-4 週）

#### 1.1 理解汽車業的客源結構

**客源的六大來源**

| 來源 | 特點 | 客源質量 | 競爭激烈度 | 適合新進業代 |
|------|------|--------|---------|----------|
| **展示室進店** | 被動客戶，已有購買意向 | 高 | 高（競爭激烈） | ✓ 優先 |
| **電話詢問** | 有明確購買需求 | 中-高 | 中 | ✓ 優先 |
| **朋友轉介** | 高信任度，轉化率高 | 高 | 低 | ✓ 適合 |
| **舊客戶回訪** | 已有購車經驗，易成交 | 高 | 低 | ✓ 適合 |
| **主動開發電話** | 需要說服，但能鍛鍊能力 | 中 | 低 | ✓ 可嘗試 |
| **社群媒體引流** | 需要長期積累，但粘性高 | 中-高 | 低 | ✓ 長期 |

**優先策略**：新進業代的前 3 個月應該集中在「展示室進店」和「電話詢問」上，掌握基本的銷售流程。同時開始準備「轉介」和「社群」的工作，為後續積累客戶。

#### 1.2 客源開發的核心方法

**方法 1：展示室進店客戶的有效接待**
- 這是最容易的客源，但也是最容易被浪費的
- 許多業代會「搶客」，導致客戶體驗差，轉化率低

**展示室進店客戶的接待流程**：
```
客戶進店
  ↓
【第 0 步】主動觀察與判斷（15 秒）
- 客戶是首購還是換購？
- 客戶對什麼車型感興趣？
- 客戶身上有什麼線索（購物袋、小孩、工作證等）？

  ↓
【第 1 步】熱誠招呼與寒暄（30 秒）
- 「歡迎光臨！請進」（親切、不唐突）
- 觀察客戶的反應，判斷是否願意交談

  ↓
【第 2 步】了解需求（3-5 分鐘）
- 「請問您今天是來看車，還是純粹瞭解一下？」
- 根據回答追問：「對什麼車型感興趣？」「用途是什麼？」「預算大概多少？」
- 關鍵：多聽，少說

  ↓
【第 3 步】匹配推薦（5-10 分鐘）
- 基於客戶需求，推薦 2-3 款車型
- 帶客戶看車，邊走邊介紹

  ↓
【第 4 步】深入商談（10-15 分鐘）
- 在展示室內，詳細介紹客戶感興趣的車型
- 確認配置、價格、優惠

  ↓
【第 5 步】推進決策（5-10 分鐘）
- 「如果今天簽下來，我們可以幫您爭取最好的優惠。您想什麼時候提車？」
- 如果客戶猶豫：「那我先幫您預訂，這樣價格能鎖定。改天您再想清楚。」

  ↓
【第 6 步】留下聯絡方式（1-2 分鐘）
- 無論成交與否，都要留下客戶聯絡方式
- 「這是我的名片，有任何問題隨時聯絡我」
```

**評估指標**：
- 進店到邀請坐下的成功率：80% 以上
- 了解到需求的成功率：70% 以上
- 成交率（首次進店 30 天內）：15-20%
- 信息留存率（留下聯絡方式）：90% 以上

**方法 3：朋友轉介的開發**
- 轉介客戶的成交率最高（50-70%）
- 需要主動向現有客戶要求轉介

**如何要求轉介**：
```
【成交後馬上要求】
「您對我們的服務很滿意，太感謝了。
其實我最希望的，就是能為您的朋友、親戚也提供一樣好的服務。
您身邊有沒有最近想換車、或有新車需求的朋友？
如果有，您可以介紹給我，我會以最好的誠意為他們服務。」

【定期回訪時要求】
「最近過得怎麼樣？車開得還好嗎？
對了，您身邊有沒有人最近在考慮買車？
如果有，我很希望能為他們服務。」

【特殊時機要求】（新年、升職、有小孩等）
「聽說您升職了，恭喜！
那麼，您身邊有沒有同事、朋友也在考慮改善用車？
我可以幫忙介紹，給你們最優惠的政策。」
```

**轉介客戶的承諾**：
- 轉介方：給予「轉介禮」或「積分獎勵」
- 被轉介方：給予「轉介客戶專屬優惠」

**方法 4：舊客戶的開發（續保和換購）**
- 舊客戶是業代最寶貴的資產
- 續保率和換購率直接影響長期業績

**舊客戶開發的三個時機**：

時機 1：**購車 6 個月後**
- 目的：確認使用情況，建立關係
- 聯絡方式：電話或線下回訪
- 內容：
  - 「親愛的客戶，您好！購車已經有段時間了，車開得怎麼樣？」
  - 「有任何問題，歡迎隨時聯絡我。」
  - 「最近有任何車輛保養需求，我也可以幫您安排。」

時機 2：**購車 2-3 年後（換購時機）**
- 目的：推薦新車，促進換購
- 聯絡方式：電話、短信、社群媒體
- 內容：
  - 「親愛的客戶，您的 Corolla 已經使用 2 年多了。」
  - 「我們最新推出的 Altis，配置和性能都升級了，特別適合您這樣的用戶。」
  - 「如果感興趣，我可以為您安排一次試乘體驗。」

時機 3：**購車 1 年後（保險續保時機）**
- 目的：續保，並推薦其他產品（保險、維修保養）
- 聯絡方式：電話或短信提醒
- 內容：
  - 「親愛的客戶，您的車輛保險即將到期，我來幫您續保。」
  - 「同時，我也可以幫您檢查一下維修保養的需求。」

**舊客戶維護的頻率**：
- 首次購車後 1 週：主動回訪
- 購車後 1 個月：詢問使用情況
- 購車後 3 個月：邀請參加廠商活動
- 購車後 6 個月：重點回訪
- 購車後 1 年：保險續保提醒
- 購車後 2-3 年：換購推薦

#### 1.3 客源開發的四週行動計畫

**第 1 週：展示室客戶接待強化**
- 目標：掌握展示室進店客戶的標準接待流程
- 活動：
  - 課堂培訓：展示室接待的 6 步流程（2 小時）
  - OJT：主管帶看 5 場展示室進店客戶（5-8 小時）
  - 自主練習：獨立接待 3 場展示室進店客戶（3-5 小時）
- 評估：
  - 能復述 6 步流程 ✓
  - 能獨立接待完整流程 ✓
  - 留存客戶聯絡方式成功率 80% 以上 ✓

**第 2 週：來電客戶的應對**
- 目標：掌握來電客戶的初期應對和轉化
- 活動：
  - 課堂培訓：來電客戶應對話術和技巧（2 小時）
  - 角色扮演：模擬 10 個不同的來電場景（2 小時）
  - OJT：聽主管應對 5 通來電，學習技巧（2-3 小時）
  - 自主練習：獨立應對 3 通來電，主管事後反饋（2-3 小時）
- 評估：
  - 能應對 5 種常見來電場景 ✓
  - 來電轉化率（來電 → 進店 / 試乘）30% 以上 ✓

**第 3 週：朋友轉介的開發和維護**
- 目標：建立轉介意識，主動向客戶要求轉介
- 活動：
  - 課堂培訓：轉介的重要性和方法（1.5 小時）
  - 實戰演練：如何自然地向客戶要求轉介（2 小時）
  - 執行：
    - 回訪 5 位現有客戶（舊客戶或同事的客戶）
    - 主動要求轉介，至少爭取 2 個轉介線索（2-3 小時）
  - 跟進：對轉介客戶進行初期接觸和邀請（2-3 小時）
- 評估：
  - 每週轉介線索 2+ 個 ✓
  - 轉介客戶的轉化率 50% 以上 ✓

**第 4 週：客源開發綜合實踐**
- 目標：綜合應用四種客源開發方法，形成業務習慣
- 活動：
  - 客源多元化：同時進行展示室客戶、來電客、轉介客、舊客戶的開發
  - 數據記錄：每天記錄客源來源、轉化進度、聯絡方式
  - 主管檢討：每週複盤，分析哪些客源質量最高
  - 績效評估：
    - 進店客戶數量：15+ 人
    - 留存聯絡方式：12+ 人
    - 進一步跟進：8+ 人
    - 初步成交 / 意向：1+ 台

**月度關鍵指標**：
- 新客源開發數：20+ 個
- 客源留存率：80%+
- 初步轉化率（開發 → 進店 / 試乘）：40%+

### 【第二模組】客戶維護與關係管理（第 5-8 週）

#### 2.1 客戶生命週期和維護策略

**客戶生命週期的五個階段**

```
             客戶價值
                │
                │    ╱╱╱
                │  ╱╱╱
                │╱╱╱
                └──────────────→ 時間
              新客 → 成長期 → 成熟期 → 衰退期 → 流失/轉介
```

| 階段 | 時間 | 特點 | 維護策略 | 目標 |
|------|------|------|--------|------|
| **新客期** | 購車後 0-3 個月 | 滿意度最高，但容易流失 | 頻繁接觸、確認滿意 | 確保滿意，積累轉介機會 |
| **成長期** | 3-12 個月 | 逐漸適應車型，產生維修保養需求 | 定期回訪、推薦服務 | 深化關係，提高粘性 |
| **成熟期** | 1-3 年 | 客戶對車輛熟悉，但可能對品牌忠誠度下降 | 主動推薦新車、保險續保 | 爭取換購、續保 |
| **衰退期** | 3 年以上 | 客戶可能考慮換車，或流向其他經銷商 | 積極推薦換購、優惠政策 | 爭取續保或換購 |
| **流失/轉介** | 已購車 3 年以上且無再購 | 客戶可能轉向競爭對手或其他經銷商 | 深度回訪、特別優惠 | 爭取重新激活或爭取轉介 |

#### 2.2 客戶維護的五個方面

**方面 1：確保購後滿意度**（新客期）

**購車後的跟進時間表**：
```
購車當天：
  ├─ 送客前：確認所有交車手續已完成
  ├─ 交車說明：詳細解釋車輛功能、保養知識
  └─ 聯絡方式：留下服務顧問的電話

購車後 3 天：
  ├─ 打電話回訪：「車開得怎麼樣？有沒有問題？」
  ├─ 提醒事項：「這個功能您會用嗎？」
  └─ 售後服務：「有任何問題，隨時聯絡我」

購車後 1 週：
  ├─ 邀請進店檢查：「新車需要進店檢查一下」
  ├─ 免費首保：「新車的首保很重要，我來幫您安排」
  └─ 驗收車況：「車有沒有出現任何問題？」

購車後 1 個月：
  ├─ 詢問駕駛體驗：「開了一個月，感受怎麼樣？」
  ├─ 了解使用情況：「有沒有習慣新車的配置？」
  └─ 推薦保養：「距離下次保養還有多遠？我可以提醒您」

購車後 3 個月：
  ├─ 定期保養：「該進行定期保養了」
  ├─ 滿意度調查：進行正式的客戶滿意度評估
  └─ 轉介邀請：「對我們的服務滿意，能介紹朋友嗎？」
```

**客戶滿意度評估**：
- 車輛質量滿意度：評分
- 購買流程滿意度：評分
- 銷售人員專業度：評分
- 售後服務滿意度：評分
- 總體滿意度：評分（目標 85 分以上）

如果滿意度低於 80 分，需要進行「挽救行動」：
- 找出具體的不滿意原因
- 立即採取改進措施（可能涉及維修、補償等）
- 2 週後重新評估

**方面 2：定期回訪與關係建設**（成長期）

**回訪的頻率和內容**：

| 時間 | 聯絡方式 | 內容 | 目的 |
|------|--------|------|------|
| 每月 1 次 | 電話或短信 | 「最近開車感覺怎麼樣？」 | 保持關係，了解近況 |
| 每季 1 次 | 電話或線下 | 保養提醒、優惠推薦 | 推薦服務，提高粘性 |
| 每年 2-4 次 | 邀請參加活動 | 廠商新車發布會、客戶感謝會 | 強化品牌印象 |
| 生日或周年 | 短信或微信 | 「祝您生日快樂」「購車一週年快樂」 | 表達關心，增加好感 |

**回訪的黃金話術**：
```
【開場】
「您好！我是小王，您的 Corolla 銷售顧問。」
「最近開車感覺怎麼樣？」

【傾聽】
「聽起來還不錯！有沒有什麼不太滿意的地方？」
「操控感如何？油耗怎麼樣？」

【提供價值】
如果沒有問題：
「那太好了。對了，您的里程數是多少了？」
「該進行定期保養了，我來幫您預約時間。」

如果有問題：
「您遇到什麼問題了？」
「沒關係，這很容易解決。我這就幫您預約服務。」

【延伸】
「對了，最近有一個客戶回訪活動，邀請您參加。」
「或者您身邊有朋友想要買車？我可以幫您介紹最好的優惠。」

【結尾】
「好的，我下次再聯絡您。有任何問題，隨時給我打電話。」
「這是我的微信/LINE，也可以直接聯絡我。」
```

**方面 3：轉介的激勵和管理**

客戶維護的最高境界是「客戶主動為你轉介新客」。

**建立轉介激勵機制**：

| 轉介等級 | 轉介人數 | 獎勵內容 | 適用客戶 |
|--------|--------|--------|--------|
| 銀牌轉介客 | 1-2 位 | 轉介禮（停車卡、行車紀錄器等） | 普通客戶 |
| 金牌轉介客 | 3-5 位 | 轉介禮 + 現金獎勵（500-1000 元） | 經常轉介的客戶 |
| 鉑金轉介客 | 5 位以上 | 現金獎勵（2000+ 元）+ 特殊服務 | 超級轉介客 |

**轉介客戶的管理**：
- 記錄「誰轉介了誰」，以及轉介客戶的成交情況
- 成交後，立即通知原轉介客戶，並送上獎勵
- 定期統計「轉介成績」，在月會上公開表揚

**方面 4：客戶數據的系統管理**

客戶維護的基礎是有一套清晰的客戶管理系統。

**客戶檔案應該包含的內容**：
```
基本信息：
  ├─ 姓名、性別、年齡、職業
  ├─ 聯絡方式（電話、微信、郵箱、LINE）
  ├─ 地址、工作地點
  └─ 特殊信息（生日、結婚紀念日、孩子生日）

購車信息：
  ├─ 購車日期、車型、顏色、配置
  ├─ 價格、付款方式
  ├─ 保險、貸款信息
  └─ 提車日期、交車里程

交互記錄：
  ├─ 購後回訪記錄（日期、內容、反饋）
```

#### 3.1 社群媒體在汽車銷售中的角色

**為什麼要做社群媒體？**

傳統模式：
```
銷售代表 ← → 客戶（一對一）
```

社群媒體時代：
```
銷售代表的社群 ← → 粉絲/客戶（一對多）
```

**三大好處**：
1. **建立個人品牌** — 不是「為公司賣車」，而是「為個人品牌賣車」
2. **持續觸達客戶** — 不需要每次都主動打電話，客戶會主動看你的動態
3. **吸引潛在客戶** — 優質的內容會自然吸引有購車需求的人

**社群媒體的投資回報率（以 1 年為例）**：
```
投入：每天 30 分鐘發文、互動 → 每年 182.5 小時

潛在回報：
  - 粉絲增長：500-1000 人（取決於內容質量）
  - 粉絲中購車意向者：50-100 人
  - 轉化成交：10-20 台（每台業績 50-100 萬）
  - 年度業績增加：500-2000 萬

投資回報率：遠高於冷電話營銷
```

- 已經有明確的購車需求或購車意向
- 質量高、轉化率最高（30-50%）

**為什麼來電客最有價值？**

| 特點 | 原因 | 對業代的意義 |
|------|------|-----------|
| 主動聯絡 | 客戶已經做了購車決策，只是在比較 | 轉化率高 |
| 時間寶貴 | 打電話說明客戶很認真，不是「隨便問問」 | 成交可能性大 |
| 容易失去 | 客戶可能同時聯絡多家經銷商 | 快速反應很重要 |
| 信息豐富 | 通過電話可以獲得大量的需求信息 | 可以精準推薦 |

**來電客轉化漏斗**：
```
100 通來電
    ↓ (接通率 80%)
80 通接通
    ↓ (應對成功率 70%)
56 位客戶了解了需求
    ↓ (邀請進店成功率 50%)
28 位客戶進店
    ↓ (成交率 40%)
11 台成交

計算：100 通來電 × 80% × 70% × 50% × 40% = 11 台成交
每個環節的優化都能倍增最終成交數
```

#### 4.2 來電客應對的五個環節

**環節 1：快速接通（第一通電話）**

**時間限制**：
- 最好在 2 秒內接通（不要超過 5 秒）
- 如果不能立即接通，也要在 1 分鐘內回電

**接電話的語氣**：
```
✓ 親切、熱誠、有精神
✗ 冷淡、不耐煩、好像在忙

標準開場：
「感謝來電！我是 Toyota 展示室的業務代表小王。請問您今天怎麼稱呼？」

核心目標：讓客戶感覺到「被重視」
```

**第一個 10 秒要做什麼**：
1. 報上自己的名字和公司
2. 熱誠招呼
3. 詢問客戶姓名
4. 準備好紙筆記錄客戶信息

---

**環節 2：深度傾聽和需求了解（電話的前 3-5 分鐘）**

**三個關鍵問題**：
```
問題 1：「請問您今天來電是想了解哪款車？」
目的：了解客戶的初步意向

問題 2：「您選擇這款車的原因是什麼？」
目的：了解客戶的核心需求（空間、安全、油耗、配置？）

問題 3：「什麼時候您會做購車決策？」
目的：了解客戶的時間緊迫性
```

**傾聽的技巧**：
- 不要急著推銷，先了解需求
- 用「嗯」「我理解」等語句表示在認真聽
- 間隔 10-15 秒提出下一個問題，給客戶說話的空間
- 記錄客戶的每一個重要信息

**記錄模板**：
```
客戶姓名：        性別：        年齡：
聯絡方式：        郵箱：
客戶需求：
  - 想要的車型：
  - 用途：
  - 預算：
  - 對車的需求點：

購車時間：        購車意向度（1-10）：
競爭對手：（是否在比較其他品牌？）
```

---

**環節 3：匹配推薦和價值傳達（電話的 5-10 分鐘）**

**推薦策略**：
- 基於客戶的需求，推薦 1-2 款最合適的車
- 不要推薦太多選項，會讓客戶難以決策
- 強調這款車「為什麼最適合他」

**推薦話術**：
```
「根據您剛才說的需求，我覺得 Altis 最適合你。」

為什麼？因為：
1. 您需要空間大的車 → Altis 的軸距比 Corolla 長 100mm，空間寬敞
2. 您注重安全性 → Altis 配備了最新的Toyota Safety Sense，主動安全配置業界最全
3. 您在意油耗 → Altis 的平均油耗 15.8km/L，比同級車省 10%

所以，Altis 真的最符合您的所有需求。」
```

**避免的話術**：
```
✗ 「我們的車最好」— 客戶不相信
✗ 「其他品牌都不行」— 顯得缺乏自信
✗ 「現在買有優惠」— 容易被認為是銷售套路
✓ 「根據你的需求，這款車最合適」— 以客戶為中心
```

---

**環節 4：異議處理和推進（電話的 10-15 分鐘）**

**常見異議和應對**：

異議 1：「我還要再想想」
```
回應：「完全理解。那我想了解一下，您還有什麼顧慮嗎？
是配置不夠？還是想對比一下其他品牌？」

（找出真實的顧慮，才能有效回應）

可能的真實顧慮：
  - 價格太貴 → 介紹優惠和貸款方案
  - 想對比其他品牌 → 比較差異，強調 Toyota 的優勢
  - 不確定是否合適 → 邀請試乘體驗
```

異議 2：「我要跟家人商量一下」
```
回應：「很好，這是重要的決定。
那這樣，我幫您預約一個時間，你和家人一起來試乘，怎麼樣？
試乘沒有任何壓力，純粹讓你們親身體驗一下車的性能。
這週或下週哪天更方便？」

（把「想想」轉化為「具體行動」）
```

異議 3：「別的經銷商價格更便宜」
```
回應：「我理解。那我能問一下，他們報的是什麼配置的車？
是否包含交車、保修、保險？」

（比較「全方位成本」，而不只是購車價）

「其實如果把所有成本算進來，我們的方案通常更划算。
而且，我們的售後服務也更完善。」
```

---

**環節 5：邀請進店和確認承諾**

**邀請進店的話術**：
```
「根據我們剛才的討論，Altis 確實很適合您。」

「為了讓您更真切地了解這款車，我想邀請您到我們的展示室試乘一次。」

「試乘很簡單，就是讓您親自開一開，感受一下操控和空間。
試乘後，您會更了解這款車是否真的適合您。」

「這週五下午或週六上午，您有時間嗎？
我會親自帶您體驗，還會為您詳細介紹所有的優惠政策。」
```

**確認承諾**：
- 確認試乘的日期和時間
- 確認客戶的聯絡方式
- 告訴客戶提前多久來（方便停車、時間充裕）
- 提醒客戶帶上駕照

**電話結束前的確認**：
```
「好的，我確認一下：
你叫 [客戶名]，聯絡方式是 [電話]，
我們約在這週五下午 2 點，你來試乘全新 Altis，對嗎？

太好了。我會在試乘前 1 小時短信提醒你。
如果有任何問題，隨時可以聯絡我。
我的電話是 09XX-XXXXX，微信是 xxx。

非常期待這週五與你見面！」
```

  - 進店客戶的試乘完成率 90% 以上 ✓
  - 成交 1+ 台 ✓

**第 12 週：來電客經營綜合評估**
- 目標：總結來電客經營的經驗，形成長期策略

    - 邀請進店率
    - 進店試乘率

  - 瓶頸分析：找出每個環節的薄弱點，制定改進計畫
  - 經驗總結：記錄成功案例和失敗案例，建立自己的「來電應對心法」
  - 後續規劃：制定來電客經營的長期目標和日常工作流程
- 評估：
  - 12 週來電客總數：50+ 通 ✓

當這四個環節都做好時，你的業績將不再是「偶發的成交」，而是「穩定的、持續的增長」。
