## 🎓 新增知識點

### L0.5 級分類原理
```
問題：為什麼需要 L0.5 級？
答：前面 L0（ETF）和後面 L1-L4（規則匹配）之間
   存在「最高優先級」的數據源空隙

解決：引入 L0.5 級
   = 官方 TWSE/TPEX API 的直接映射
   = 完全避免任何規則匹配的偏差
   = 99.9% 準確度保證
```

### 非阻塊式初始化
```
原理：JavaScript 異步執行

loadLatestIndustryData()  ← 後台非阻塊執行
  ├─ 發送 HTTP 請求
  ├─ 等待 API 響應（~2-3 秒）
  ├─ 構建 COMPANY_INDUSTRY_MAP
  └─ 可用

用戶操作              ← 前台正常進行
  ├─ 頁面加載（無延遲）
  ├─ 開始查詢（不等待）
  └─ 結果展示

結果：兩者並行進行，用戶無感
```
