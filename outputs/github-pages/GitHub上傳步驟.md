# GitHub Pages 部署步驟

完成後你的網址會是：https://neferyang.github.io/stock-tool/

---

## 步驟一：建立 Repository

1. 開啟瀏覽器，登入 https://github.com（用你的帳號 neferyang）
2. 點右上角「+」→「New repository」
3. Repository name 填：`stock-tool`
4. 選「Public」（必須，GitHub Pages 免費版需要 Public）
5. 勾選「Add a README file」
6. 按「Create repository」

---

## 步驟二：上傳 index.html

1. 進入剛建立的 `stock-tool` repository
2. 點「Add file」→「Upload files」
3. 把 `index.html` 拖進去（就在這個資料夾裡）
4. 下方 Commit changes 說明可隨意填，按「Commit changes」

---

## 步驟三：開啟 GitHub Pages

1. 在 repository 頁面，點上方「Settings」
2. 左側選單找「Pages」
3. Source 選「Deploy from a branch」
4. Branch 選「main」，資料夾選「/ (root)」
5. 按「Save」

---

## 步驟四：取得網址

- 約等 1~2 分鐘，重新整理 Settings → Pages 頁面
- 會看到「Your site is live at https://neferyang.github.io/stock-tool/」
- 這個網址就是你的工具，手機、電腦、平板都能用
- 貼到痞客邦部落格的文章裡，讀者也能開！

---

## 之後要更新工具怎麼做？

直接在 GitHub 上點 `index.html`→「Edit」→修改→「Commit changes」，約 1 分鐘後自動更新。
