## 🐛 已知限制

```
1. 首次加載時有 1-2 秒延遲（初次 TWSE API 請求）
2. 後續搜尋瞬間完成（本地緩存）
3. 英文公司名稱不支持（API 無英文名）
   例如：輸入「Apple」無法搜尋到 AAPL
   解決：直接輸入代碼「AAPL」或中文「蘋果」（需 KEYWORD_MAP）
```
