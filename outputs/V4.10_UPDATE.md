# v4.10 更新說明 - 當日走勢改為 5 日備選

**版本**：v4.10  
**日期**：2026-06-04  
**改進**：當日走勢優化（日線備選）

---

## 📊 用戶期待 vs 現實

### 用戶期待
```
當日按鈕 → 黑色背景分鐘線走勢（像證券商 app）
  ├─ 09:00-13:30 詳細分鐘級走勢
  ├─ 開盤、最高、最低、收盤
  └─ 實時更新
```

### 現實（數據源限制）
```
Yahoo Finance 提供：
  ✓ 日線 K 線（開高低收）
  ✓ 日成交量
  ✗ 分鐘級走勢（需付費 API 或專業版）
```

---

## ✅ 當前解決方案

**當日按鈕行為**：
```
用戶點擊「當日」按鈕
  ↓
系統檢查數據源
  ↓
無分鐘線數據可用
  ↓
自動顯示 5 日日線走勢
（包含最近 5 天含當日的完整走勢）
```

**顯示內容**：
- 📈 5 天日線 K 線（開高低收）
- 📊 MA20、MA60 均線
- 📉 日成交量

**當前應用選擇**：Yahoo Finance（無依賴、完全免費）

## 🎯 當日走勢使用指南

**查詢當日走勢**：
1. 輸入股票代碼（如 2330）
2. 點擊「當日」按鈕
3. 圖表顯示最近 5 天的日線走勢
4. 可看到當日與前 4 天的價格走勢對比

**優勢**：
- ✅ 看清當日整體走勢
- ✅ 看到週期背景（與前 4 天對比）
- ✅ 無需外部 API，完全免費

**限制**：
- ⚠️ 不是分鐘級（盤中實時）
- ⚠️ 是日線級（一天一根蠟燭線）
