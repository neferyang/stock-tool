## 🎯 估值判斷標準

### 合理價值區間

```
P/E 估值區間 = fairPE × 0.7 ~ fairPE × 1.3

示例（以半導體業 fairPE 25x）：
├─ 低估帶：P/E < 25 × 0.8 = 20x          [買進機會]
├─ 合理帶：P/E 20-32.5x                  [持有區間]
└─ 高估帶：P/E > 25 × 1.3 = 32.5x        [減碼觀望]
```

### 特殊邏輯（景氣循環股）

```
逆向操作邏輯（航運、鋼鐵、塑膠、化學）：

景氣狀態 ←→ P/E表現 ←→ 操作策略
───────────────────────────────
景氣高峰  → P/E極低   → 減碼賣出（獲利不可延續）
景氣末期  → P/E升高   → 逐步減碼
景氣谷底  → P/E高企   → 布局買進（市場過度悲觀）
景氣回升  → P/E降低   → 加碼買進

備選指標：股價淨值比（P/B）1.0x 作為底部支撐
```

## 📈 更新影響分析

### 對估值結果的影響

**提升評價的產業**：
- 汽車工業 (+50%)：10x → 15x，更符合整車大廠實際估值
- 電子零組件 (+20%)：15x → 18x，反映車用/AI供應鏈成長
- 鋼鐵工業 (+25%)：8x → 10x，符合常態運行區間

**調降評價的產業**：
- 半導體業 (-17%)：30x → 25x，反映晶圓代工重資本特性
- 通信網路 (-41%)：22x → 13x，修正為公用事業合理水準
- 數位雲端 (-6%)：32x → 30x，微幅調整至中點

**買點建議變化**：
```
以台積電 (2330) 為例，目前 P/E 28x：

舊估值（fairPE 30x）：
- 合理區間：21-39x → P/E 28x 為低估 ✓ 買進訊號

新估值（fairPE 25x）：
- 合理區間：17.5-32.5x → P/E 28x 為合理偏高 → 持有訊號

→ 結論變化：「買進機會」→「持有並觀察」
```
