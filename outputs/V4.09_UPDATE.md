## 🔧 核心改進

### 問題診斷
當日走勢圖顯示的當日數據不正確，主要原因：
- Yahoo Finance 返回的是日線 K 線數據
- 當日的 K 線在盤中時可能是不完整的（未收盤）
- 某些情況下當日數據與前日相同，但成交量為 0

### 解決方案

**數據過濾優化**：
```javascript
// 新的驗證規則
const idx = ts.map((_,i)=>i).filter(i=>{
  if(closes[i] == null || closes[i] <= 0) return false;
  
  // 檢查是否是無效的當日數據
  // 若價格與前日相同且成交量為 0，表示盤中未結束
  if(i > 0 && closes[i] === closes[i-1] && (!vols[i] || vols[i]===0)) 
    return false;
    
  return true;
});
```

**當日顯示邏輯優化**：
```javascript
if(n === -1) {
  // 檢查最後一根 K 線有效性
  if(endIdx > 0 && 數據不完整) {
    s = Math.max(0, AP.length - 5);  // 顯示 5 根（更安全）
  } else {
    s = Math.max(0, AP.length - 2);  // 顯示 2 根（當日+前日）
  }
}
```
