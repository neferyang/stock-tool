## 📝 驗證方法

1. **手動測試法**（推薦用於初期驗證）
   ```
   1. 打開應用頁面
   2. 輸入股票代碼（如 2330）
   3. 點擊「查詢」
   4. 檢查產業分類結果是否正確
   5. 記錄分類結果和置信度
   ```

2. **批量測試法**（後續可自動化）
   ```javascript
   // 在瀏覽器控制台執行
   for (const code of ['2330', '2207', '1101', ...]) {
     const result = classifyIndustry(code+'.TW', '', '', '');
     console.log(code, result);
   }
   ```
