## 📊 升級概述

## 🔌 API 集成方案

### **方案一：靜態本地映射（推薦初期）**
- **優點**：快速、無網路依賴、性能最佳
- **缺點**：需手動更新官方變更
- **實施時間**：已完成

### **方案二：動態 OpenAPI 整合（推薦長期）**
```javascript
// 初始化時自動抓取最新官方分類
async function loadLatestIndustryData() {
  try {
    // 上市公司
    const twseUrl = "https://openapi.twse.com.tw/v1/opendata/t187ap03_L";
    const twseData = await fetch(twseUrl).then(r => r.json());
    
    // 上櫃公司
    const tpexUrl = "https://www.tpex.org.tw/openapi/v1/mopsfin_t187ap03_O";
    const tpexData = await fetch(tpexUrl).then(r => r.json());
    
    // 合併並建立映射表
    const allCompanies = [...twseData, ...tpexData];
    window.COMPANY_INDUSTRY_MAP = new Map(
      allCompanies.map(item => [item.公司代號.trim(), item.產業別])
    );
    
    console.log('✅ 官方產業分類已同步');
  } catch (e) {
    console.warn('⚠️ 無法獲取官方最新分類，使用本地映射');
  }
}
```

**優點**：
- 永遠與官方同步
- 自動捕捉新增企業和產業變更
- 完全透明可溯源

**缺點**：
- 需網路連線
- 首次載入可能稍慢（~2-3 秒）
- 官方 API 故障時需備選方案
