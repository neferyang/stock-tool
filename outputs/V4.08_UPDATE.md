## 🎯 核心改進

### 1. 加入當日走勢 ✨

**新功能**：
- 新增「當日」按鈕
- 顯示最近 5 天走勢（包含當日）
- 用戶可快速查看當日的開盤、最高、最低、收盤

**按鈕順序**：
```
當日 | 1個月 | 3個月 | 6個月 | 1年 | 全部
```

**應用場景**：
```
用戶想看當日走勢 → 點擊「當日」按鈕
→ 圖表顯示最近 5 天的走勢
→ 含 MA20、MA60、成交量
→ 快速判斷當日強弱
```

### 2. 移除公告卡片

**原因**：
- 公告列表展示邏輯複雜
- 無法完整和準確地呈現
- 用戶更需要核心估值功能

**說明**：
- 後端公告獲取邏輯保留（未來可重新啟用）
- 應用界面更清爽
- Focus 在核心功能上
