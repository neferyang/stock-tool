## 工具 3：來電客應對記錄表

用途：記錄每通來電的客戶信息、需求、應對過程、結果等

```
來電客應對記錄表

【來電基本信息】
來電時間：2026/6/12 下午 2:15
來電時長：8 分 30 秒
接電人：小王
來電方式：直接來電展示室

【客戶信息】
客戶姓名：黃先生
性別：男
年齡：約 40-50 歲（推測）
聯絡方式：0912-XXXX
職業：（未詢問，後續跟進）
初步印象：聲音有禮貌，說話清晰，似乎有明確購車計畫

【需求分析】
所述需求：
- 想要一輛「家庭用車，空間大，安全性好」
- 預算：「大概 100-120 萬」
- 用途：「家裡有 2 個小孩，需要比較寬敞的空間」
- 購車時間：「這個月或下個月」

深度需求分析：
- 核心痛點 1：空間不足（現在開 Corolla，孩子坐著不舒服）
- 核心痛點 2：安全性（有 2 個小孩，特別重視安全）
- 核心痛點 3：省油（家庭預算考慮）

推薦車型：Altis 或 Sienta（需要進一步了解是否想要 MPV 的靈活性）

【應對過程記錄】

| 時間 | 內容 | 評估 |
|------|------|------|
| 2:15-2:20（5 分） | 開場招呼，確認客戶名字和需求 | ✓ 親切、專業 |
| 2:20-2:30（10 分） | 了解客戶的具體需求（家庭、小孩、預算） | ✓ 傾聽充分 |
| 2:30-2:35（5 分） | 根據需求推薦 Altis（空間大、安全配置完善、油耗省） | ✓ 匹配度高 |
| 2:35-2:40（5 分） | 介紹 Altis 的核心優勢（軸距長、Toyota Safety Sense、16km/L 油耗） | ✓ 強調客戶關注點 |
| 2:40-2:43（3 分） | 處理客戶異議（「我想再看看其他品牌」） | ✓ 有效回應，強調 Toyota 優勢 |
| 2:43-2:45（2 分） | 邀請進店試乘 | ✓ 成功邀約 |

【異議分析】
客戶說：「我想先看看其他品牌，比如 Nissan，價格好像更便宜」

我的回應：
「我理解。Nissan 的車確實價格有競爭力。
不過呢，如果我們把空間、安全配置、油耗這些你關心的要素比較，
Toyota Altis 在同級車中是最全面的。
而且，Toyota 的保值率最高，以後賣車也更值錢。
我建議你試乘一次 Altis，親身體驗一下，再和其他品牌比較。」

評估：✓ 效果不錯，客戶沒有再提出強烈反對

【邀請進店結果】
邀請是否成功：✓ 成功
試乘時間：下週五（6/18）下午 3 點
客戶確認：「好的，我會準時來」

【後續行動計畫】
| 行動 | 時間 | 負責人 | 備註 |
|------|------|-------|------|
| 短信提醒 | 6/17 下午 | 小王 | 「親愛的黃先生，明天下午 3 點見！」 |
| 進店接待 | 6/18 下午 3 點 | 小王 + 主管陪伴（OJT） | 重點：展示空間、安全配置、油耗體驗 |
| 試乘陪伴 | 6/18 下午 3:30 | 小王 | 讓客戶感受操控性和空間舒適度 |
| 後續跟進 | 6/19 或 6/20 | 小王 | 詢問試乘反饋，推進成交談判 |

【評估與反思】
本通來電的應對評分：85 / 100

做得好的地方：
- ✓ 快速接通、親切招呼
- ✓ 傾聽充分，了解客戶真實需求
- ✓ 推薦的車型與客戶需求匹配度高
- ✓ 有效處理「比較其他品牌」的異議
- ✓ 成功邀約進店

可以改進的地方：
- 可以問更多關於「現在開什麼車」的問題，了解客戶的換車原因
- 可以預估試乘時間、提醒帶駕照、停車位置等細節
- 可以先預約「預約優惠」，增加客戶的期待感

下一步：
- 準備好 Altis 的試乘車，確保車況整潔
- 準備好「試乘優惠清單」和「Altis vs Nissan 對比表」
- 通知洗車部門，6/18 前確保試乘車狀態最佳
```
