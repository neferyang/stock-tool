# Toyota 經銷商教育訓練資料庫

## 用途

這個 skill 是一套完整的 Toyota 經銷商教育訓練資料庫系統。它的核心功能是：

1. **生成教育訓練講義** — 根據使用者需求（課程主題、目標學員、訓練等級）快速產出教案大綱、完整講義或 PPT 教材
2. **組織與檢索教育資料** — 整理、分類、查詢現有的教育訓練資料，協助你快速找到相關教材
3. **標準化訓練課程** — 確保跨據點的訓練內容一致，建立教育訓練的資料庫和知識庫

## 資料庫結構

skill 的資料庫涵蓋以下三大類別：

### 1. 業代教育（業務代表訓練）
- **新進業代教育**：包含 TWSM（豐田銷售方式）、商品知識、客戶關係、銷售技巧、法務財務、保險知識等全方位新人訓練課程
- **中級業代教育**：提升業代的銷售能力、客戶經營策略、CR（客戶關係）深化課程
- **嘉信新進業代教育**：專針對和泰嘉信據點的新進業代訓練教材，包含分期租賃、保險專業等
- **主題課程**：包括商品知識補充、服務知識、業務實務、一般常識、汽車安全知識等

### 2. 主管教育（管理層訓練）
- **新任課長教育**：針對新上任課長的管理基礎課程，涵蓋主管責任、團隊合作、顧客滿意、管理理念等
- **營業所管理教育**：營業所長級別的經營管理課程，包含時間管理、PDCA、販賣活動管理
- **儲備課長教育**：課長晉升前的準備課程，含 TWSM、時間管理、管理工具等
- **補充資料**：管理哲學、故事教學、激勵內容等支援教材

### 3. 初級 OJT（現場訓練）
- 配合新進人員進場訓練的實務教材
- 結合業代教育教材進行現場實踐指導

## 使用方式

### 場景 1：快速生成教訓練講義

當你說「幫我產出新進業代的 TWSM 訓練講義」或「我需要一份主管時間管理的教案大綱」時，skill 會：

1. 識別你需要的課程類別（業代 / 主管 / OJT）
2. 確認學習等級和詳細程度（概要、完整講義、PPT 簡報版）
3. 根據資料庫內容，提取相關教材內容，結構化成你需要的格式
4. 輸出為：教案大綱（markdown 或 text）、完整講義（markdown 或 HTML）、或 PPT 格式

**範例輸出格式：**
- **教案大綱**：課程目標、主要單元、重點內容、時間配置
- **完整講義**：含詳細說明、案例、練習題、補充資料
- **PPT**：簡潔的簡報版本，適合教室講授

### 場景 2：查詢與檢索現有教育資料

當你說「我想找 TWSM 相關的教材」或「查詢所有新進業代的課程清單」時，skill 會：

1. 根據關鍵字、課程類別、難度等級進行搜尋
2. 列出相關的教材清單、簡介、和資源位置
3. 告訴你哪些教材可用、更新狀態、建議搭配使用的其他資料

### 場景 3：課程設計與教育計畫

當你說「我想設計新進業代的三個月訓練計畫」時，skill 會：

1. 根據不同月份和學習進度，建議課程組合
2. 提供各課程的時間建議、教學方式、評估方法
3. 確保課程之間的邏輯順序和進度安排

## 主要課程模組

skill 內建的主要課程包括（以下為簡化清單，實際資料庫遠更豐富）：

### 業代課程
- TWSM（豐田銷售方式）5P 架構
- 客戶關係管理（CR）：客戶分級、續保、開拓
- 商品知識：各車款特色、規格、賣點
- 銷售技巧：應對、商談、成交、客戶異議處理
- 法務與稅務：汽車買賣、稅務申報、法律責任
- 保險知識：汽車保險類型、理賠、要保書填寫
- 服務知識：售後服務、維修保養
- 業務實務：目標計畫、業績管理、獎勵制度

### 主管課程
- 新任課長管理基礎
- PDCA 循環管理
- 時間管理與工作效率
- 團隊建設與合作
- 顧客滿意度管理
- 販賣活動與業績促進
- 豐田管理理念與 TPS

### OJT 課程
- 展示室值班應對
- 新車交車說明
- 中古車查估基礎

## 操作步驟

### 要求 1：生成教訓練講義

```
請求：我需要一份「新進業代客戶關係管理」的教案大綱，時間 2 小時。
```

Skill 會：
1. 讀取現有的 CR 相關教材（客戶分級、續保、開拓的 PPT）
2. 提取核心內容，重組成 2 小時的教案結構
3. 提供：教案大綱、建議教學活動、時間配置、補充資料清單

### 要求 2：查詢教育資料

```
請求：請列出所有關於「TWSM 豐田銷售方式」的教材清單。
```

Skill 會：
1. 搜尋資料庫中所有含 TWSM 的教材
2. 列出檔案名稱、課程等級、格式、簡述
3. 建議：可搭配使用的課程模組、學習順序

### 要求 3：產出 PPT 格式教材

```
請求：根據主管教育「時間管理」課程，產出 30 分鐘的 PPT 簡報版本。
```

Skill 會：
1. 抓取「時間管理」課程的核心內容
2. 簡化為適合 PPT 簡報的形式（大標題、要點、視覺化）
3. 輸出為 PPT 檔案，可直接在會議上播放

## 重要提醒

- **資料完整性**：skill 內含 300+ 個訓練教材檔案，覆蓋業代、主管、OJT 三大類別
- **課程搭配**：某些課程建議與其他課程組合學習，skill 會自動提出配置建議
- **版本管理**：不同時期的教材可能有新舊版本，查詢時會提示最新版本
- **客製化調整**：如果你對 skill 產出的內容需要調整（刪除、補充、改進），直接告訴我，我會協助修改

## 輸出格式選項

根據你的需要，skill 可以產出以下格式：

1. **教案大綱** (Markdown / Text)
   - 課程目標、主要單元、時間配置、教學要點
   - 適合講師備課、快速審閱

2. **完整講義** (Markdown / HTML)
   - 詳細的講授內容、案例、練習、補充資料
   - 適合學員自學、課後複習

3. **PPT 簡報** (.pptx / .ppt)
   - 精簡的投影片版本
   - 適合課堂講授、內部分享會

4. **課程清單** (文字 / 表格)
   - 詳細的教材清單、分類、連結
   - 適合規劃訓練計畫

## 快速開始

1. **告訴我你的訓練需求**
   - 「我需要一份新進業代的商品知識教案」
   - 「列出所有主管管理課程」
   - 「產出 TWSM 的 PPT 版本」

2. **我會確認細節**
   - 課程等級（新進、進階、專家）
   - 訓練時長（30 分鐘、1 小時、半天、全天）
   - 輸出格式（大綱、講義、PPT、清單）

3. **Skill 產出教材**
   - 提供所需的教訓練內容
   - 附上補充資料和教學建議
