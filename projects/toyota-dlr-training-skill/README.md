## 🎯 Skill 說明

### 名稱
`toyota-dlr-training-skill`

### 功能
Toyota 經銷商教育訓練資料庫與講義生成工具

### 主要用途
1. **快速生成教訓練講義** — 根據課程主題、目標學員、訓練時長，自動產出教案大綱、完整講義或 PPT 簡報
2. **組織與檢索教育資料** — 查詢和檢索 300+ 個教育訓練教材，按課程類別、等級、主題分類
3. **規劃訓練計畫** — 建議課程組合、進度安排、評估方法

### 涵蓋範圍

**業代教育**
- 新進業代基礎訓練（TWSM、商品知識、客戶關係、銷售技巧、法務財務、保險知識等）
- 中級業代進階課程
- 嘉信新進業代專門教材（分期租賃、保險等）
- 各車款商品知識（ALTIS、CAMRY、YARIS、Prius、Previa、Innova 等）

**主管教育**
- 新任課長管理基礎課程
- 營業所管理教育（時間管理、PDCA、販賣活動管理）
- 儲備課長晉升訓練
- 豐田管理工具與理念

**初級 OJT**
- 新進人員現場訓練指引
- 展示室應對、交車說明、中古車查估

**補充資料**
- 管理故事與案例
- 激勵與態度教材
- Toyota 企業文化與哲學

## 🚀 使用方式

### 快速開始

直接告訴我你的訓練需求，skill 會自動觸發：

```
「我要訓練新進業代，需要一份『TWSM 豐田銷售方式』的 2 小時教案大綱。」

「列出所有關於『客戶關係管理（CR）』的教育訓練教材。」

「我需要『時間管理』的 30 分鐘 PPT 簡報版本。」
```

### 輸出格式

- **教案大綱** (Markdown) — 課程目標、主要單元、時間配置、教學要點
- **完整講義** (Markdown/HTML) — 詳細的講授內容、案例、練習、補充資料
- **PPT 簡報** (框架文件) — 精簡的投影片版本設計
- **課程清單** (文字/表格) — 教材清單、分類、連結

## 🤔 常見問題

**Q: 如何調整生成的教案內容？**
A: 直接告訴我想修改的部分（刪除、補充、重組），我會協助調整。

**Q: 資料庫內的教材可以拿來直接教學嗎？**
A: 可以。Skill 提取的是現有教材的內容，生成的教案可直接在課堂使用。

**Q: 找不到需要的課程怎麼辦？**
A: 查閱 training-database.md 的完整清單。如仍未找到，我可根據現有教材組合新課程。

**Q: 輸出格式可以轉換嗎（如 PDF、Word）？**
A: 可以。Skill 預設輸出 Markdown 或 PPT 框架，我可協助轉換為其他格式。

**Q: 能否根據我的據點需求客製化課程？**
A: 可以。提供你的需求（人員等級、訓練目標、時間限制），我會客製化課程內容。
