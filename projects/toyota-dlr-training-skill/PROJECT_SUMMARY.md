## 🎯 專案目標

建立一套完整的 **Toyota 經銷商教育訓練資料庫 Skill**，用於：
1. **快速生成教訓練講義** — 根據需求自動產出教案大綱、完整講義、PPT 簡報
2. **組織與檢索教育資料** — 管理 300+ 個教育訓練教材，支援快速查詢
3. **標準化訓練課程** — 確保跨據點訓練內容一致，建立知識庫

## 🚀 使用方式

### 快速開始

只要告訴我你的訓練需求，Skill 會自動觸發：

```
「我要訓練新進業代，需要一份『TWSM 豐田銷售方式』的 2 小時教案大綱。」
↓
Skill 自動生成：課程目標、3 個主要單元、時間配置、教學活動、補充資料
```

### 三種主要使用場景

**場景 1：生成教訓練講義** ✨
```
輸入：課程主題 + 目標學員 + 訓練時長 + 輸出格式
輸出：教案大綱 / 完整講義 / PPT 簡報框架
範例：「新進業代、2 小時、TWSM、教案大綱」
```

**場景 2：查詢教育資料** 🔍
```
輸入：關鍵字 / 課程類別 / 難度等級
輸出：教材清單、簡述、版本標示、搭配建議
範例：「列出所有 CR 相關教材」
```

**場景 3：規劃訓練計畫** 📋
```
輸入：訓練等級 / 訓練週期 / 目標人群
輸出：課程組合建議、進度規劃、評估方法
範例：「設計新進業代的 3 個月訓練計畫」
```

## 📞 支援與維護

### 常見問題

**Q: 能否調整生成的教案內容？**
A: 完全可以。直接告訴我想修改的部分，我會協助調整。

**Q: 資料庫的教材可以直接教學嗎？**
A: 可以。Skill 提取的是現有教材內容，生成的教案適合直接在課堂使用。

**Q: 如何添加新的教材到資料庫？**
A: 把新教材放在教育資料夾，告訴我位置，我會更新資料庫清單。

**Q: 能否生成其他格式（PDF、Word 等）？**
A: 預設是 Markdown 和 PPT 框架，我可協助轉換為 PDF、Word 或其他格式。
