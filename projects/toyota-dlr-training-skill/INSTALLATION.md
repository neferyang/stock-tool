## Skill 功能總結

### 1. 教訓練講義生成
- 輸入：課程主題、目標學員、訓練時長、輸出格式
- 輸出：結構化的教案大綱、完整講義、或 PPT 簡報框架

**範例使用**
```
我要訓練新進業代，需要一份『TWSM 豐田銷售方式』的 2 小時教案大綱。
請告訴我課程目標、主要單元、時間配置、重點內容和教學建議。
```

### 2. 教育資料查詢與檢索
- 輸入：關鍵字、課程類別、難度等級
- 輸出：教材清單、簡述、資源位置、搭配建議

**範例使用**
```
我想查詢所有關於『客戶關係管理（CR）』的教育訓練教材清單。
請列出所有相關的課程、按等級分類、並告訴我哪些是最新版本。
```

### 3. 訓練計畫設計
- 輸入：訓練等級、訓練週期、目標人群
- 輸出：課程組合建議、進度規劃、評估方法

**範例使用**
```
我想設計新進業代的 3 個月訓練計畫。
請建議第一個月應該上什麼課、第二個月、第三個月，
以及每個月的評估方式。
```

## 最佳實踐

### 使用 Tip 1：明確指定需求
說得越具體越好。例如：
- ✓ 「新進業代、2 小時、TWSM、教案大綱格式」
- ✓ 「主管、時間管理、30 分鐘 PPT、含講師提示」

而不是：
- ✗ 「給我教育訓練資料」
- ✗ 「做一份講義」

### 使用 Tip 2：善用搜尋功能
當你不確定教材名稱時，直接用關鍵字搜尋：
- 「列出所有 CR 相關教材」
- 「查詢保險知識的課程」
- 「顯示所有新任課長教育材料」

### 使用 Tip 3：參考課程組合建議
Skill 內建的課程組合建議（在 training-database.md 中）可作為訓練計畫設計的參考基礎。

## 常見問題

**Q: 如果我想調整生成的教案內容？**
A: 直接告訴我你想修改的部分（刪除、補充、重組），我會協助調整。

**Q: 資料庫內的教材可以拿來直接教學嗎？**
A: 可以。Skill 提取的是現有教材的內容，你可以直接在課堂使用生成的教案。

**Q: 如果找不到我需要的課程怎麼辦？**
A: 查詢 training-database.md 中的完整清單。如果仍未找到，我可以根據現有教材組合出新的課程。

**Q: 輸出格式可以轉換嗎（如 PDF、Word）？**
A: 可以。Skill 預設輸出 Markdown 或 PPT 框架，但我可以協助轉換為其他格式。
