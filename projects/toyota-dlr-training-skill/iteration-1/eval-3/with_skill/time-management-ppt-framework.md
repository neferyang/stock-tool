# 時間管理 PPT 簡報框架（30 分鐘版）
## 新任課長班訓練教材

**課程目標** | 讓新任課長學會用「優先順序管理」和「時間區塊法」有效安排工作，提升團隊管理效率
**目標學員** | 新任營業課長、儲備課長
**訓練時長** | 30 分鐘（含互動）
**教室規模** | 15-30 人
**教材格式** | 講師投影版 + 講師提示卡

---

## 完整 PPT 框架設計（12 頁）

### 第 1 頁：開場 & 課程目標（2 分鐘）

**頁面標題**
```
時間管理：課長的第一堂課
```

**重點項目**
- 「一個課長的時間 = 整個營業所的時間」
- 今天學到什麼？
  - 時間問題的根源診斷
  - 優先順序管理框架
  - 實戰應用技巧

**視覺化建議**
- 背景：深藍色（Toyota 企業色）+ 簡潔白色文字
- 左側：放一個「時鐘」圖示，指針指向 12 點
- 右側：放課程三個重點（用三個不同顏色的方塊展示）
- 底部：放課程時間「30 分鐘」

**講師提示卡**
```
開場語：
「各位新任課長，恭喜你們上任了。從今天開始，你的每一分鐘都不再只是你自己的時間，
而是整個團隊的時間。一個課長如果時間管理得好，團隊效率就會跟著提升；反之則會成為
整個營業所的瓶頸。今天我們要學的，就是如何做一個『會時間管理的課長』。」

互動提示：
讓每位新課長舉手說「你現在最頭痛的工作問題是什麼？」（約 1-2 個人回答）。
然後說：「其實你們的問題，99% 都是『時間管理』的問題。」
```

**時間配置** | 2 分鐘

---

### 第 2 頁：故事導入 - 課長的一天（2.5 分鐘）

**頁面標題**
```
你的一天是這樣嗎？
```

**重點項目**
- 早上 8:30：檢查昨日業績報表 ❌
- 9:00：主管突然召集開會
- 9:30：有客戶投訴，需要立即回應
- 11:00：才發現部屬的銷售報表還沒審核
- 12:00：午餐時間，但要邊吃飯邊回客戶信件
- 下午忙著「救火」...
- 5:00 下班時，發現計畫中的「團隊培訓」完全沒做

**視覺化建議**
- 用時間軸（horizontal timeline）展示一天的工作流
- 每個時間點用不同顏色標示（紅色：被動反應 / 黃色：計畫中但被打斷 / 綠色：完成計畫工作）
- 右側放一個「疲憊課長」的人物插圖（簡筆畫風格）
- 強調「全天都在『救火』，沒有時間做『重要的事』」

**講師提示卡**
```
講述語：
「這個故事，是不是你？很多新任課長都是這樣過日子的。每天都忙，但到了月底一看，
該完成的工作沒有完成，該達成的目標沒達成。最糟的是，部屬們看你那麼忙，
也跟著開始『忙著救火』，整個營業所就陷入了『瞎忙』的惡性循環。」

引導問題：
「請問，這一天裡，有多少時間是『課長自己決定要做什麼』？」
讓學員回答，通常答案會是「幾乎沒有」。

重點：「沒錯！因為所有時間都被『別人的急事』搶走了。」
```

**時間配置** | 2.5 分鐘

### 第 3 頁：診斷 - 時間都去哪了？（2 分鐘）

**頁面標題**
```
時間問題的四大根源
```

**重點項目**
1. **被動反應型工作**（突發客訴、主管交辦）
   - 占時間比例：60-70%
   
2. **沒有優先順序**（什麼都要做，什麼都要做好）
   - 占時間比例：20-30%
   
3. **缺乏授權與委派**（什麼都自己做）
   - 占時間比例：10-15%
   
4. **計畫不清楚**（沒想好一天要做什麼）
   - 占時間比例：100%（貫穿所有問題）

**視覺化建議**
- 用環形圖表（pie chart）或堆疊長條圖展示時間占比
- 用不同顏色區分四個根源
- 在中心放「課長」的圖示，被四個根源包圍
- 加上箭頭和百分比，讓數字視覺化

**講師提示卡**
```
診斷說明：
「我們現在來看看，為什麼時間會這麼緊張。根據我們對營業所的調查，
新任課長通常有這四大時間黑洞...」

重點強調：
「看到沒？被動反應就占了 60-70% 的時間！這意味著，你每天有接近 7 成的時間，
都不在自己的掌控範圍內。這就是為什麼你覺得『好忙，但什麼都沒做』。」
```

**時間配置** | 2 分鐘

### 第 4 頁：核心概念 - 優先順序矩陣（3 分鐘）

**頁面標題**
```
Eisenhower 優先順序矩陣
重要 vs. 緊急
```

**重點項目**

| | **緊急** | **不緊急** |
|---|---|---|
| **重要** | **象限 I**<br/>危機管理<br/>→ 立即做<br/>（客戶投訴、重大問題） | **象限 II**<br/>戰略發展<br/>→ 規劃做<br/>（部屬培訓、流程改善） |
| **不重要** | **象限 III**<br/>時間浪費<br/>→ 能免則免<br/>（無關的會議、閒聊） | **象限 IV**<br/>逃避行為<br/>→ 不做<br/>（滑手機、閒散活動） |

**視覺化建議**
- 用大的 2×2 矩陣，每個象限用不同顏色
  - 象限 I（紅色）：緊急且重要 
  - 象限 II（綠色）：重要但不緊急
  - 象限 III（黃色）：緊急但不重要
  - 象限 IV（灰色）：既不重要也不緊急
- 在象限 II 放一個「⭐」符號，強調這是課長應該花時間的地方
- 右側放一個範例：把之前第 2 頁「課長的一天」中的各項工作分別放入各象限

**講師提示卡**
```
核心概念說明：
「這個矩陣是時間管理的黃金法則，叫『Eisenhower 矩陣』。
它把所有的工作分成四個象限...」

逐象限解釋：
- 象限 I（紅色，緊急+重要）：「這是危機，必須立即處理。但我們的目標是減少象限 I 的工作。」
- 象限 II（綠色，重要但不緊急）：「這才是課長應該花大部分時間的地方！為什麼？
  因為象限 II 的工作，做好了，就能預防象限 I 的危機產生。培訓部屬、建立流程、
  規劃業績...都在這裡。」
- 象限 III（黃色，緊急但不重要）：「騙你的時間。別人著急，不代表你的事重要。
  要學會說『我先去處理更重要的事，等等再幫你』。」
- 象限 IV（灰色，既不重要也不緊急）：「這是逃避。偶爾放鬆可以，但不要讓它占據你的時間。」

課長的使命：
「優秀的課長，應該把 60% 的時間花在象限 II。這樣，象限 I 的危機就會自然減少。」
```

**時間配置** | 3 分鐘

### 第 5 頁：案例應用（3 分鐘）

**頁面標題**
```
案例 1：客戶投訴 vs. 部屬培訓
我該做什麼？
```

**重點項目**

**情景**：
- 下午 2 點，一個老客戶因為車的問題來投訴
- 同時，你原定 2 點要與部屬小王進行「銷售技巧 1 對 1 教練」

**三種常見反應**
1. ❌ **放棄培訓，全力應對客訴**
   - 結果：客戶問題解決了，但部屬沒得到成長
   - 長期後果：員工銷售能力沒提升→月底業績達不到
   
2. ❌ **繼續培訓，忽視客訴**
   - 結果：客戶失望，可能流失
   - 長期後果：客戶滿意度下降→月度 CS 扣分
   
3. ✅ **優先順序管理的做法**
   - 先用 5 分鐘了解客訴內容（判斷危急程度）
   - 如果能當場解決→快速解決，再回到培訓
   - 如果需要複雜處理→指派信任的老員工先處理，約定 30 分鐘後跟進
   - 回到與小王的培訓，完成計畫
   - 下班前 1 小時再深入處理客訴細節

**視覺化建議**
- 用三個不同顏色的流程圖展示三種做法
- 在第三種做法上，用綠色勾勾✅標示「正確的優先順序」
- 在左側用「時間秤」圖示，顯示「象限 I vs. 象限 II」的平衡

**講師提示卡**
```
案例導入：
「現在我們用一個真實的例子來練習。假設這個情況發生在你身上...」

逐一分析三種做法的後果，強調：
「看起來，第三種做法好像很耗時。但想想看：
- 如果每個月都有這樣的『放棄培訓』事件，那一年下來，部屬就沒有得到任何培訓。
- 結果就是部屬的銷售能力停滯，業績上不去。
- 然後你要花更多時間去『救火』。
- 最後陷入惡性循環。

相反，如果你今天花 30 分鐘找到『兼顧的方法』，
長期來說，你能減少 50% 的『救火』時間。」

互動提示：
「你們遇過類似的情況嗎？誰來說說，當時你怎麼處理的？」
讓 2-3 位新課長分享他們的案例。
然後用優先順序矩陣來分析他們的做法，是屬於哪一象限。
```

**時間配置** | 3 分鐘

### 第 6 頁：實戰工具 1 - 時間區塊法（2.5 分鐘）

**頁面標題**
```
每週一次「時間規劃」
用 30 分鐘改變一週
```

**重點項目**

**做法**：每週日晚上或週一早上，花 30 分鐘做以下規劃：

1. **列出本週的所有工作**（5 分鐘）
   - 例：業績追蹤、客訴處理、團隊會議、部屬一對一、流程改善提案、月度報表
   
2. **分類到優先順序矩陣**（10 分鐘）
   - 哪些是象限 I（危機）
   - 哪些是象限 II（戰略）
   - 哪些可以減少或委派
   
3. **在週曆上「區塊化」安排**（15 分鐘）
   - 每項重要工作配置具體時間段
   - 象限 II 的工作一定要有固定時段（例：週三下午 2-3 點是「部屬培訓時間」，不動搖）
   - 為象限 I 的「意外狀況」預留 20% 的機動時間

**視覺化建議**
- 展示一份「改革前」和「改革後」的週曆比較
- 改革前：亂七八糟，到處都是臨時會議和客訴
- 改革後：有清晰的時間區塊，象限 II 的工作有固定時段
- 用不同顏色區分不同類型的工作（例：紅色=象限 I，綠色=象限 II，灰色=機動時間）
- 放一個「30 分鐘」的時鐘圖示，強調只需花 30 分鐘規劃

**講師提示卡**
```
導入方式：
「知道了優先順序很重要，那我們要怎麼實際應用呢？
最簡單、最有效的工具，就是『時間區塊法』。」

詳細說明：
「原理很簡單：不是『找到時間』去做象限 II 的工作，
而是『創造時間』、『保護時間』。怎麼做？就是在週曆上，
用『區塊』來標示每項重要工作的時間。」

實際示範：
「比如說，你決定『每週三下午 2-3 點，我要和部屬進行銷售教練』。
這個時間就是『固定的區塊』，除非天塌下來，否則不動搖。
這樣，即使有其他事情發生，部屬至少還是能保證每週有一次培訓。」

強調 30 分鐘的價值：
「這個規劃只需要 30 分鐘，但它能幫你節省一整週的時間浪費。
投資回報率是 1:14。」

互動提示：
「誰能說出，你現在每週一定要做的『象限 II 工作』有哪些？」
讓 2-3 位新課長舉例，然後說「好，回家以後第一件事，
就是在你的週曆上，用『區塊』把這些時間標出來。」
```

**時間配置** | 2.5 分鐘

### 第 7 頁：實戰工具 2 - 說「不」的智慧（2 分鐘）

**頁面標題**
```
如何優雅地拒絕不重要的工作
```

**重點項目**

**情景**：
- 同事：「課長，要不要一起參加這個會議？」
- 主管：「課長，能不能幫忙整理這份報表？」
- 員工：「課長，你有空聊一下嗎？」

**三句黃金拒絕詞**

1. **「我現在正在執行更重要的工作，能不能稍後？」**
   - 適用於：象限 III 的「緊急但不重要」工作
   - 技巧：說出你現在做的是什麼（讓對方知道你有優先順序）

2. **「這個工作很適合 XXX 去做，我建議找他。」**
   - 適用於：可以委派的工作
   - 技巧：不是推卸，而是「最適化分配」

3. **「我先記下來，看看這週有沒有時間。」**
   - 適用於：確實不是現在的優先順序
   - 技巧：禮貌但果決，並且給出「我會考慮」的信號

**視覺化建議**
- 用三個不同的對話框展示三句金句
- 每句話旁邊標示「適用情況」和「效果」
- 用綠色勾勾✅標示「這樣說是職業的」
- 對比紅色叉叉❌標示「不要這樣說」（例：「我沒時間」、「我不管」）

**講師提示卡**
```
導入方式：
「實行優先順序管理，最難的一步是什麼？『說不』。
新任課長通常很害怕拒絕別人，怕被認為不夠配合、不夠熱情。
但其實，『說不』是管理者必須學會的技能。」

詳細說明三句金句，並給出實際例子：

金句 1：「我現在正在執行更重要的工作，能不能稍後？」
「這句話聽起來硬，但其實是在『教育』對方：
課長的時間是有優先順序的，不是所有事都一樣急。
對方聽到這句話，就會開始思考『我的事重不重要』。」

金句 2：「這個工作很適合 XXX 去做，我建議找他。」
「這不是推卸責任，而是『最適化分配』。
你是課長，你的責任是確保工作完成，不一定要自己做。
如果能讓合適的部屬做，他們還能得到成長。一舉兩得。」

金句 3：「我先記下來，看看這週有沒有時間。」
「誠實但有風度。這樣說，對方不會覺得被拒絕，
但他也會明白『可能沒有時間』。大部分情況下，他會自己去找其他人。」

警告：
「千萬別說『我沒時間』。為什麼？因為『沒時間』意味著你的時間管理不好。
要說『這不在我的優先順序裡』，這樣才專業。」

互動提示：
「現在我們一起練習。我扮演你的同事，說『課長，我有點事想請教你』。
你們用剛剛學的三句金句之一來回應。」
選 2-3 位新課長實況練習。
```

**時間配置** | 2 分鐘

### 第 8 頁：案例故事 - 一個課長的轉變（2.5 分鐘）

**頁面標題**
```
故事案例：小陳課長的 3 個月
```

**故事內容**

**第一個月**：小陳剛上任，很積極。
- 每天從早到晚都在忙，應對客訴、參加各種會議、自己寫報表
- 月底業績目標達成了，但「部屬培訓」項目完全沒推進
- 員工們看小陳那麼忙，也都跟著焦躁不安

**第二個月**：主管對小陳說：「你部屬的銷售能力沒有提升。」
- 小陳意識到問題，開始嘗試「優先順序管理」
- 每週日晚上，花 30 分鐘規劃週曆
- 把「部屬一對一教練」列為「固定時間」（象限 II）
- 開始學會說「這個工作我現在沒時間，可以先交給 ○○ 試試」

**第三個月**：變化出現了。
- 部屬的銷售技巧有明顯提升，業績貢獻度提高 15%
- 客訴數量反而下降了（因為問題被提前預防）
- 小陳每天下班時間比以前早 1 小時，但工作反而更有成效
- 團隊的工作氣氛也變好了（因為領導者不再一直在「救火」）

**啟示**：
- 優先順序管理，需要「一點勇氣」和「一點堅持」
- 不是一下子就完美，而是逐月改善
- 三個月後，你會發現自己變成了一個更有效率的課長

**視覺化建議**
- 用「三個月時間線」展示小陳的轉變
- 第一個月：用紅色表示「忙亂」（時間軸混亂）
- 第二個月：用黃色表示「開始調整」（出現區塊）
- 第三個月：用綠色表示「有效率」（清晰的區塊）
- 在每個月旁邊放「關鍵行動」的標籤
- 右側放「成果數據」：員工銷售提升 15%，客訴下降 X%，下班早 1 小時

**講師提示卡**
```
故事導入：
「現在我想跟你們分享一個真實的故事。這是一個真實發生過的課長轉變。
他叫小陳，三個月前和你們一樣，剛上任課長。」

逐月講述，在關鍵點暫停：

第一個月後：
「一個月過去，小陳的業績達成了。但他發現一個問題...」
（暫停，讓學員猜測：「什麼問題？」）
「他意識到，他花了這麼多時間『救火』，根本沒有時間去『培訓部屬』。
而沒有部屬的成長，長期來說，業績是上不去的。」

第二個月：
「小陳開始改變。他做了一個簡單的動作：每週日晚上，用 30 分鐘規劃週曆。
他把『部屬教練』標在週三下午 2 點，一週 1 小時。
剛開始很難，因為他要學會拒絕別人。但他堅持了。」

第三個月：
「轉折來了。部屬開始變聰明了。為什麼？因為他們每週都有一對一教練。
他們的銷售能力提升，能獨立處理更多客戶。
這反而減少了小陳的『救火』時間。
而且，因為問題被提前預防了，客訴數量也下降了。」

最後：
「三個月後，小陳變成了一個什麼樣的課長呢？
他每天仍然很忙，但忙得有成效。
他下班時間比以前早，員工們也不再那麼焦躁。
最重要的是，他建立了一個『會自己成長的團隊』。」

互動提示：
「你們三個月後，想變成什麼樣的課長？小陳的案例，是不是你想要的？」
讓 1-2 位新課長回答。
```

### 第 9 頁：互動練習 - 你的優先順序是什麼？（2 分鐘）

**頁面標題**
```
練習時間（2 分鐘）
你的優先順序是什麼？
```

**練習說明**

在這 2 分鐘，每位新課長要在紙上寫下：
1. **你這週最重要的「象限 II 工作」是什麼？**
   - （例：部屬培訓、流程優化、新商品學習）
   
2. **你打算怎麼「保護」這個時間？**
   - （例：固定在週幾、幾點到幾點？有什麼方式能確保不被打斷？）

**講師說明**
「別急著回答，想清楚。這不是考試，是你對自己的承諾。」

**視覺化建議**
- 在投影片上放一個「工作表樣本」，讓學員參考
- 左側：「象限 II 的工作」和「我的選擇」
- 右側：「我的時間保護計畫」
- 底部：倒數計時器，提醒「2 分鐘」（實際可用簡單的沙漏圖示）

**講師提示卡**
```
練習導入：
「現在輪到你們。我給 2 分鐘，讓你們寫下來：
第一，你認為你這週最重要的『象限 II 工作』是什麼？
第二，你打算怎麼在週曆上『保護』這個時間？」

說明規則：
「別想太複雜。就一個工作，一個時間段。
比如說，『週三下午 2 點，與小王進行銷售教練』，就可以了。」

時間提醒：
「2 分鐘開始...（1 分鐘時提醒一次）...還有 30 秒...」

完成後：
「寫完了沒？好，這張紙別丟掉。帶回家，貼在你的辦公桌上。
下個禮拜的今天，我們再聚在一起，你要告訴我：『我有沒有執行這個計畫？』」
```

**時間配置** | 2 分鐘

### 第 10 頁：常見問題 & 解答（2 分鐘）

**頁面標題**
```
Q&A：你可能有的疑問
```

**常見問題與回答**

**Q1：「我的工作本來就很難預測，怎麼可能提前規劃？」**
- A：「沒有 100% 的預測，但 60-70% 的工作是可預測的。
  用區塊法規劃這 60-70%，剩下的 30-40% 用『機動時間』來應對。
  這樣也比『全部都亂來』好。」

**Q2：「象限 I 的危機真的來了，我難道就要放著不管？」**
- A：「當然不是。但首先要『判斷』：這真的是危機嗎？
  很多『自以為是危機』的事，其實不用你親自處理。
  學會委派，讓信任的部屬去處理，你在 30 分鐘後再跟進，就夠了。」

**Q3：「我怕拒絕別人，被認為不夠 team player。」**
- A：「相反！一個真正的 team player，是『把事情做對』，不是『凡事都答應』。
  如果你說『好，我來做』，但最後因為時間不夠做得很爛，
  那才是不 team player。誠實地說『我現在沒時間做好』，
  反而是更專業的表現。」

**Q4：「我試過規劃，但還是會被打亂。」**
- A：「很正常。改變需要時間。3 週是養成習慣的最小周期。
  前兩週會很難，第三週開始就會變輕鬆。
  我們建議：用『試用期心態』，先堅持 3 週，然後評估成效。」

**視覺化建議**
- 用「Q&A 卡片」風格展示，每個問題一張卡片
- 問題用紅色、答案用綠色
- 在回答旁邊加上「tips」圖示或「燈泡」圖示
- 留出視覺空間，顯示這是「開放式的互動」

**講師提示卡**
```
互動方式：
「在我講 Q&A 之前，有人要先問問題嗎？」
（等待 10-15 秒，看有沒有人舉手）

如果有人提問，就回答他的問題，然後說：
「很好的問題。其他人也可能有同樣的疑惑，我們一起看看...」

如果沒人提問，就說：
「好，那我先預測你們可能會有的問題...」

逐一回答，重點強調：
- Q1：時間管理不是 100% 預測，而是 60% 預測 + 40% 應變
- Q2：區分「真危機」和「假危機」很重要，關鍵是『委派』
- Q3：拒絕不是不配合，而是『更專業的選擇』
- Q4：改變需要 3 週，鼓勵他們『試用 3 週』再評估

結尾：
「這些都是真實的挑戰。下個月我們會再聚在一起，
你們要告訴我：『實行這三個禮拜，遇到什麼困難？』
我們一起想辦法解決。」
```

**時間配置** | 2 分鐘

### 第 11 頁：三個行動承諾（2 分鐘）

**頁面標題**
```
帶走三個承諾
從今天開始改變
```

**三個行動承諾**

**承諾 1：這個禮拜**
- [ ] 用 30 分鐘做一次「週曆規劃」
- [ ] 在週曆上用「區塊」標出至少 2 個「象限 II」的工作
- [ ] 執行：不動搖這些區塊

**承諾 2：這個月**
- [ ] 每週都做「週曆規劃」（變成習慣）
- [ ] 至少「拒絕」或「委派」掉 3 項「象限 III」的工作
- [ ] 評估：你的「下班時間」有沒有提早？

**承諾 3：三個月內**
- [ ] 完成「部屬培訓計畫」（至少進行 12 次一對一教練）
- [ ] 觀察：部屬的「自主性」和「銷售能力」有沒有提升？
- [ ] 反思：你的「團隊效率」有沒有改善？

**視覺化建議**
- 用「三層樓梯」或「三個里程碑」展示三個時間段
- 每個承諾用不同顏色標示（例：1 週=黃色，1 月=橙色，3 月=綠色）
- 在每個承諾下放 2-3 個具體的「檢查項目」
- 右側放一個「日曆」或「進度條」圖示

**講師提示卡**
```
導入方式：
「在我們結束前，我要請你們做三個承諾。不是對我的承諾，
而是對你自己、對你的團隊的承諾。」

逐一講述三個承諾，並強調時間框架：

承諾 1（這個禮拜）：
「先從小開始。這個禮拜的任務很簡單：
用 30 分鐘規劃週曆，標出兩個『象限 II』的工作，然後執行。
你會驚訝，只是這個簡單的動作，就能讓你的一週變得不一樣。」

承諾 2（這個月）：
「到了月底，我要看你的進度。
你做了幾次週規劃？你拒絕了多少個『象限 III』的工作？
最重要的是，你下班有沒有比以前早？」

承諾 3（三個月）：
「三個月後，我們會再聚在一起。那時，我想看到的，
是你的部屬真的在成長，你的團隊真的在改變。
不是因為你有多聰明，而是因為你把『象限 II 的工作』堅持做好了。」

最後的話：
「這三個承諾，不難。難的是『堅持』。
但我保證，如果你堅持，三個月後，你會很慶幸當初做了這個決定。」
```

**時間配置** | 2 分鐘

### 第 12 頁：結尾 & 鼓勵（1 分鐘）

**頁面標題**
```
一個課長的時間
就是整個團隊的未來
```

**重點項目**
- 你的時間決定你的團隊的時間
- 你選擇做什麼，就會形塑團隊的文化
- 優先順序管理，不是讓你「更忙」，而是讓你「更有成效」

**結尾語**
```
「各位新任課長，

從今天起，你們不再是『忙碌的個人』，而是『領導團隊的管理者』。
你們每做出的『優先順序選擇』，都會影響整個營業所。

這 30 分鐘，我們學到的不只是方法，更是一個想法的改變：
從『忙著救火』，轉變為『主動預防』。

回去以後，別急著改變一切。
先做好『這個禮拜』的三個承諾。
然後，一週一週、一月一月地堅持。

三個月後，你會發現自己已經變成了一個不一樣的課長。
你的部屬也會發現，他們的課長變得不一樣了。

祝各位，都能成為『會時間管理的課長』。

如果在實行中有任何困難，記得一點：『3 週是改變的最小周期』。
前兩週會很辛苦，第三週開始就會變輕鬆。

加油！」
```

**視覺化建議**
- 背景：深藍色漸層到白色（象徵從「忙亂」到「清晰」）
- 中央：大型文字「一個課長的時間 = 整個團隊的未來」
- 左側：一個向下的箭頭（「救火狀態」）
- 右側：一個向上的箭頭（「戰略狀態」）
- 底部：一個「開始」按鈕或「行動」的視覺（例如腳印或起點線）

**講師提示卡**
```
結尾語氣：
- 溫暖而堅定
- 不是「命令」，而是「邀請」和「鼓勵」
- 強調「改變很難，但值得」

最後的互動：
「在我們結束前，我想聽聽你們。
誰願意站起來說：『我承諾，從這個禮拜開始做時間規劃』？」

（讓 2-3 位新課長舉手發言，鼓勵他們說出來）

公開承諾很重要，因為：
1. 它增加了承諾的力量
2. 建立了同儕之間的相互鼓勵
3. 講師能看到誰是「真的想改變」的人

結束時：
「謝謝各位的參與。下個月我們再聚在一起，
我很期待聽你們分享『時間管理』的成果。
記住：30 分鐘規劃，改變一整週。」
```

**時間配置** | 1 分鐘

目的：讓學習從「理論」變成「實戰」

讓新課長站起來公開承諾「我這週要做週曆規劃」
目的：增加「心理承諾」的力度，提高執行率

## 使用注意事項

### 講師備課提示
1. **預留機動時間**：實際課程中，某些互動可能比預期長。建議預留 3.5 分鐘機動時間（見時間配置表）
2. **個案故事準備**：準備 1-2 個自己親眼看到的「課長時間管理失敗案例」，會比通用故事更有說服力
3. **週曆圖表準備**：帶一份你自己的週曆範本，展示「怎麼樣才是好的規劃」
4. **提問設計**：第 2、5、11 頁的互動環節，提前設計好「開放式提問」，避免冷場

### 課後追蹤
- 第一週：確認新課長有沒有做「週曆規劃」
- 第二週：詢問「執行中遇到什麼困難」
- 第三週：開始看到「改變的跡象」
- 第四週（一個月）：進行「成果檢視會」，分享彼此的進度
