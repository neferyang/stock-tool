# 5 年歷史 EPS 數據獲取指南

## 問題
以下股票的 5 年 EPS 數據呈現**完美直線增長**（過於規則），疑似不是真實數據：
- **1101**（台泥）
- **2603**（長榮）
- **2201**（裕隆）

需要從可靠來源獲取真實的 5 年 EPS 數據。

---

## 📊 推薦數據源

### 1️⃣ TWSE 公開資訊觀測站（最推薦）

**網址**：https://mops.twse.com.tw/mops/web/t05_query

**步驟**：
1. 進入網站
2. 搜尋股票代號（例如 1101）
3. 查看「財務報表」→「個股月營收」或「年度財務」
4. 提取 EPS 欄位（每股盈餘）
5. 記下 2021-2025 年度的 EPS 值

**優點**：
- ✅ 官方數據，最可靠
- ✅ 包含完整年度數據
- ✅ 可查詢財報公告日期
- ✅ 數據與股市同步更新

**範例**：
```
1101 台泥：
  2021: EPS = ?
  2022: EPS = ?
  2023: EPS = ?
  2024: EPS = ?
  2025: EPS = 5.87（當前值）
```

---

### 2️⃣ 公司年度報告

**來源**：各公司官網投資人關係部門

**步驟**：
1. 訪問公司官網
2. 進入「投資人關係」或「財務資訊」
3. 下載最新年度報告 PDF
4. 在財務表中查找 EPS（每股盈餘）
5. 逐年對比

**優點**：
- ✅ 完整財務信息
- ✅ 包含管理層討論
- ✅ 可驗證審計意見

---

### 3️⃣ 台灣證交所公開資訊平台

**網址**：https://opendata.twse.com.tw/

**方法**：
1. 下載「上市公司基本資料」
2. 查詢年度財務指標
3. 提取歷史 EPS 數據

---

## 📝 填寫修正表

編輯 `eps-correction-template.json`，填入真實的 EPS 數據：

```json
{
  "corrections": {
    "1101": {
      "historicalEPS": [
        {
          "year": "2021",
          "eps": 3.92,      // ← 實際值（需要驗證）
          "source": "TWSE MOPS"
        },
        {
          "year": "2022",
          "eps": 4.41,
          "source": "TWSE MOPS"
        },
        // ... 依此類推
      ]
    }
  }
}
```

## 💡 提示

- **交易日期**：財報通常在月初 10 天內公佈，可在 MOPS 查詢確切公告日
- **季度 vs 年度**：使用年度（Annual）EPS，不要用季度數據
- **單位確認**：台灣股票 EPS 單位為「元」，確認無誤
- **審計意見**：優先選擇通過審計（無保留意見）的財報
- **復權調整**：如涉及配股、股票分割，需確認 EPS 已復權
